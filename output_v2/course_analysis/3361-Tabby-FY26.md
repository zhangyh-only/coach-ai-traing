## 【销售技巧与方法论】

### 技巧/SOP：体验式个性化连接（"发挥个性服务的精神 / 展现工艺"）
- **所属Coach方法论**：无具名缩写；属触感体验/价值传递阶段的体验式销售
- **适用行为类型与渠道**：质感自用型、都市悦己型；Retail 正价店
- **适用环节**：触感体验 / 价值传递 / 连带推荐
- **操作步骤/要点（原文）**：
  1. 对想要独一无二、手工制作质量的顾客，**强调每件单品的特殊性与主题图案营造的个人化风格**；
  2. **展现工艺**：从刺绣到可爱微章、再到压印在皮革上的短迷字词，把这些特别细节指给顾客看；
  3. **与顾客分享，看看他们对哪个细节最有兴趣，然后询问他们为什么会特别喜欢这个细节**（用提问把产品讲解转成顾客的自我表达）。
- **应对的场景**：顾客对"经典款会不会太普通"、想要个性化/独特性的需求
- **话术原文**：「对于想要拥有独一无二、手工制作质量的顾客来说，涂鸦印花是最佳选择！」「看看他们对哪个最有兴趣，然后询问他们为什么会特别喜欢这个细节。」

## 【客户心理与沟通原则】

### 消费心理 / 驱动机制
- **质感/手工价值导向**：被"现代经典、手工质量、独一无二"打动（质感自用型）。
- **趋势/自我表达导向**：被 Y2K 丹宁、纽约街区、西部风、超现实扭曲等主题吸引（都市悦己型）。
- **场景化选择**：按"大型包/小型晚出包/日常通勤"选尺寸，决策与使用场景绑定。

### 品牌价值传递方式
- 用时装秀首发 + WWD/Marie Claire 媒体背书塑造"现代经典 + 趋势引领"的双重叙事，而非单纯讲贵。
- 引用："Coach 的春季系列是现代风格、轻松、带回忆、手工制作的氛围、风格清新、真诚回归经典。"—WWD

## 【客户问题与标准解答】
**Q**：Chain Tabby 能装下我的 iPad / 电脑吗？
**话术要点**：三种尺寸——小号容小型 iPad、中号容中型 iPad、大号(28)可容 13 寸笔记本；按使用场景选。

**Q**：Knit Soft / Monochrome 怎么保养？
**话术要点**：Knit Soft 为羊毛混纺**只能干洗**；Monochrome 染色金属配件易刮伤需小心持用。

- **Coach 专有术语对照**：Soft Grain Leather=柔软颗粒纹皮革；Bombé=立体凸饰边；Pavé C=密钉水晶 C 标；Signature C=双 C 标识。

- **资料特别强调**：Tabby 是 Coach 标志性明星系列；"丰富皮革+尺寸"满足不同顾客。
