## 【销售技巧与方法论】

### 技巧一：活人感互动（聆听 → 回应 两步法）
- **所属Coach方法论**：CoachX 无届体验 — Gen-Z 沟通 / 活人感互动（标签 "Skill"）
- **适用行为类型与渠道**：Gen-Z 顾客通用（最贴近都市悦己型/质感自用型）；全渠道
- **适用环节**：迎宾破冰 / 需求诊断（倾听需求、建立真诚连接的环节）
- **操作步骤（课程原文）**：
  1. **Step 1 聆听**：捕捉冰山下的「感受、观点、期待」
  2. **Step 2 回应**：验证听到的「感受 / 观点 / 期待」是否准确
- **回应句式模板（课程原文）**：
  - 感受：「我能感受到你……」
  - 观点：「我能感受到你（认为/觉得）……」
  - 期待：「我能感受到你（期待/希望/想要）……」
- **关键原则**：**允许留白**（回应后给对方留出空间，不急于填满，标注于流程图右下角问号图标处）
- **应对的场景**：当 Gen-Z 顾客表达一段带情绪/价值观的话时，导购需识别其"感受—观点—期待"三层并复述验证
- **话术原文（示例对话）**：见【话术库】

### 技巧二：Gen-Z 正向反馈 — 夸夸 SBI 技巧
- **所属Coach方法论**：CoachX 无届体验 — SBI（Situation 情景 / Behavior 行为 / Impact 影响）正向反馈（标签 "Skill"）
- **适用行为类型与渠道**：带教/团队管理场景为主（给同事/下属正向反馈）；通用
- **适用环节**：售后维系 / 团队带教（也可迁移到对顾客的正向肯定）
- **操作步骤（课程原文）**：
  1. **Situation 情景**：双方都知晓情景时**可简短或省略**（例："你刚刚 / 你休个假回来 / 最近 / 你昨天发的小红书…… / 省略情景"）
  2. **Impact 影响 — 展现你观点的夸夸**：**带有观点的肯定，不仅停留在事实**
     - 事实（仅陈述，不够）："最近你 NPS 分数挺高的""这月你业绩全店第一"
     - 观点（要加上）："状态不错!""真棒!""藏不住的优秀!!""太细节了!"
- **核心要点**：夸赞要从"陈述事实"升级为"表达观点+情绪"，让被夸者感到被真正看见
- **夸夸改造示例（课程原文）**：
  - 改造前（仅事实）：「这个月 tabby 卖的很好啊」
  - 改造后（事实+观点）：「这个月 tabby 卖的很好啊，继续努力呀，"tabby 大将军"!!」
- **话术原文**：见【话术库】

### 技巧三：Gen-Z 改进型反馈 — SBIR（B 问 B 答）
- **所属Coach方法论**：CoachX 无届体验 — SBIR（Situation 情景 / Behavior 行为 / Impact 影响 / Request 请求）改进型反馈；课程标注核心在 **"B 之 B 问 B 答"**（在 Behavior 行为环节用提问代替指责）（标签 "Skill"）
- **适用行为类型与渠道**：带教/团队管理场景为主（给同事指出待改进行为）；通用
- **适用环节**：团队带教 / 售后复盘
- **操作步骤（课程原文，四段式 + 行为环节双问）**：
  1. **Situation 情景**：客观描述场景（例："跟客人时在后面看手机""我看到你在接待刚才那组客人时"）
  2. **Behavior 行为 — 提问①（澄清并回应）**：用提问澄清对方行为背后的原因，而非直接批评（例："一直低头在看手机，发生啥了，是不是遇到什么事情了呀？" / 补充："我能理解线上有很多顾客消息需要回复"）
  3. **Behavior 行为 — 提问②（共创行动）**：与对方一起商量改进方案（例："那我们一起想想，如果以后再遇到类似的情况，我们怎么处理比较好？"）
  4. **Impact 影响**：说明该行为对结果的影响（例："不过现场的客人看到我们在看手机，感受会不好，也容易收到 NPS 差评。"）
  5. **夸夸（注入信心）**：以鼓励收尾，给予信心（例："太好了，这样做的话大家都开心，我完全相信你再遇到一定可以做好!"）
- **核心要点**：改进反馈不直接下结论/批评，而是"先澄清原因（提问）→ 共创解决方案（提问）→ 说明影响 → 注入信心"，保护 Gen-Z 的"安全感"需求
- **话术原文**：见【话术库】

## 【客户心理与沟通原则】

### Gen-Z（Z世代）核心消费/行为心理 — 冰山模型（Page-01）
- **冰山下（成长环境，塑造这代人的根因）**：
  - 教育资源丰富/井喷
  - 数字原住民（成长于互联网/智能手机，社媒为日常，如 LinkedIn、微信、WhatsApp 等图标所示）
  - 全球化大熔炉（"全球化大增炉"，全球视野）
- **冰山上（外显行为表现）**：
  - 5G 冲浪体验官（追求新鲜体验、信息获取快）
  - 职场行为（有自己的职场态度）
  - 情绪自由我做主（重视情绪表达自主权）
  - 真实勇敢做自己（追求真实、敢于自我表达）
- **三大核心观念/底层需求（冰山核心）**：
  - **体验感**：重视过程中的体验与感受
  - **意义感**：在意所做之事/所买之物的意义
  - **安全感**：需要被尊重、被理解、心理安全

### 沟通风格（课程主张）
- **活人感**：以真诚、有温度、不机械的方式互动，而非套路化话术
- **先聆听后回应、复述验证**：捕捉对方"感受/观点/期待"三层并确认，让对方感到被听见
- **允许留白**：给对方表达和思考的空间，不急于填满或下结论
- **夸赞要带观点+情绪**：不只陈述事实，要表达真实的赞赏与情绪（满足"被看见"需求）
- **改进反馈用提问代替批评**：以"提问澄清→共创行动→注入信心"路径给予改进建议（满足"安全感"需求）

### 沟通禁忌 / 红线（课程隐含）
- 改进反馈时**避免直接指责/下定论**（如直接说"你不该看手机"），应先提问了解原因
- 正向反馈时**避免只罗列事实而无情绪/观点**（被夸者感受不到被真正认可）
- 与 Gen-Z 沟通**避免套路化、无温度的标准话术**（违背"活人感"）

### 决策焦虑 / 内心独白（从 Gen-Z 三需求推导）
- "这个体验/这次购物对我来说有意义吗？"（意义感）
- "我会不会被当成数字或业绩，而不是被真正看见？"（体验感/被看见）
- "我表达真实想法会被尊重还是被否定？"（安全感）

### 内容偏好
- 社交媒体原住民，活跃于小红书、LinkedIn、微信等平台（举例提及"你昨天发的小红书"，可作为破冰切入点）

### 品牌价值传递方式
- Coach（CoachX 无届体验）以"读懂 Gen-Z、用活人感与真诚连接这代人"作为对年轻顾客的沟通基调，呼应品牌 Expressive Luxury（真我新奢）对"真实自我表达"的主张

**活人感互动 — 示例对话（Page-02，导购应对一位"每天早上早起化妆 2 小时全妆、不累吗"的 Gen-Z 顾客）**

**改进型反馈 SBIR（Page-03，团队带教，场景：同事接待客人时在后面低头看手机）**
- 反馈原话整段（课件"反馈原话"框）：「我能理解线上有很多顾客消息需要回复，但目前现场的顾客是非常需要关注的。之前你也因为类似情况收到过不好的 NPS 反馈，我希望这种情况不要再发生了好吗？如果你需要帮助请及时告诉我。」
- 【带教·通用·情景 S】「我看到你在接待刚才那组客人时……」

| 活人感互动要求"聆听捕捉感受/观点/期待" | 顾客表达带情绪/价值观内容时，学员的回应 | 用"我能感受到你……"句式复述并验证其感受/观点/期待中≥1层 | 不回应情绪、直接转向推销或机械应答 | 倾听共情 / 需求诊断 |

| 课程要求"允许留白" | 回应后是否给对方空间 | 不急于填满对话，给顾客表达/思考空间 | 连续输出、抢话、急于成交 | 沟通节奏 / 顾客体验 |
| 夸夸 SBI 要求"带观点的肯定，不仅停留事实" | 学员给出正向反馈/夸赞时 | 在事实基础上叠加观点+情绪（如"真棒/藏不住的优秀"） | 只陈述事实数据，无情绪无观点 | 正向反馈质量 / 团队带教 |
| SBI 情景"双方知晓时可简短或省略" | 夸赞时对情景的处理 | 视情景已知与否灵活简略，不冗余铺垫 | 机械套全四要素、啰嗦 | 反馈结构掌握 |
| 改进反馈 SBIR 要求"行为环节用提问（B问B答）" | 学员指出他人待改进行为时 | 先提问澄清原因，再提问共创方案，而非直接批评 | 直接指责/下定论（"你不该……"） | 改进反馈方式 / 心理安全 |
| SBIR 要求"说明影响 Impact" | 改进反馈中 | 客观说明该行为对顾客感受/NPS 的影响 | 只批评行为不讲影响，或夸大威胁 | 反馈有理有据 |
| SBIR 要求"以注入信心收尾" | 改进反馈结尾 | 给予鼓励、表达信任（"相信你一定能做好"） | 以批评/警告/否定结尾 | 鼓励赋能 / 心理安全 |
| 识别 Gen-Z 三大需求（体验感/意义感/安全感） | 接待年轻顾客时 | 沟通体现对其体验、意义表达、被尊重需求的回应 | 把顾客当业绩数字、套路化对待 | 顾客画像识别 / 体验 |

## 【重点提示】
- **Coach / 项目专有术语对照**：
  - **CoachX 无届体验（Beyond Experience）**：本课所属培训项目/合作方品牌，原样保留，勿翻译
  - **Gen-Z 冰山模型**：成长环境（冰山下）→ 行为表现（冰山上）→ 核心观念（体验感/意义感/安全感）
  - **活人感互动**：聆听→回应 两步法，核心是捕捉并验证"感受/观点/期待"
  - **SBI**：Situation 情景 / Behavior 行为 / Impact 影响（正向"夸夸"反馈框架）
  - **SBIR**：Situation / Behavior / Impact / Request（改进型反馈框架）；课程强调 **"B 之 B 问 B 答"**——在 Behavior 行为环节用"两次提问"（澄清并回应 + 共创行动）代替批评
- **易错/易混淆**：
  - SBI（4 步、用于夸赞）≠ SBIR（含 Request、用于改进反馈），二者均含"夸夸/注入信心"收尾，勿混淆用途
  - 本课方法论是 **SBI/SBIR 通用反馈模型**，**不是** TA Card / E.N.D. / A.L.L. / A.A.A. / 价格三明治；下游归类勿张冠李戴
  - "tabby"在本课是夸赞示例占位词，**非产品知识点**
- **红线 / 边界**：
  - 本课为沟通方法论，不含任何产品参数、价格、政策，涉及具体商品信息须以门店主数据为准，勿据本课编造
  - 改进反馈红线：勿直接指责/否定 Gen-Z（违背"安全感"需求与课程方法）
- **资料特别强调**：与 Gen-Z 沟通的底层是满足其"体验感 / 意义感 / 安全感"三需求，方法落点是"活人感、真诚、带观点的夸赞、用提问代替批评、注入信心"。
