## 产品详细介绍

### 设计灵感与结构
- **光泽皮革工艺故事（四层）**：①100% 牛皮，具有丰富天然纹理；②在意大利经过鞣制；③以自然方式进行数小时的滚筒摔软工序，呈现做旧的复古感；④最后上一层闪亮的光泽釉/釉漆。光泽皮革与麂皮都来自意大利。
- **手感与外观**：独特/耀眼的光泽、柔软的垂坠感（适合制作柔软、有垂坠度的手袋）、复古做旧质感。保有 Coach 核心**棒球手套鞣制皮革**的优美垂坠感，与 Lana & Nomad 系列单品相互呼应。
- **品牌标识**：缀有精巧的**立体雕塑 C 品牌标识**（产品图可见金色/银色立体 C logo）。
- **Juliet 结构**：简约利落的 Hobo / 新月形(弯月)廓形肩背包；配有**可拆卸的斜背带以及肩带，并以夹扣固定**。
- **配色定位**：推出更多新颜色，强调"**自然色**"——对喜欢选搭黑色手袋的人是很棒的时髦替代品；同时也有抢眼/亮眼的色彩可选。

### 核心卖点
1. **意大利原产 + 独家工艺**：100% 牛皮、意大利鞣制 + 数小时滚筒摔软 + 上釉，承载轻奢专业感与 Expressive Luxury 真我新奢的自我表达内核。
2. **复古做旧感 + 闪亮釉面**：做旧是滚筒摔软工艺特意呈现的设计风格，叠加釉面光泽，形成"既复古又耀眼"的独特视觉，呼应 GenZ 个人风格表达。
3. **自然色 = 黑色的时髦替代**：满足"习惯买黑色求稳、又想要点不一样"的顾客，给一个既安全又出彩的选色。
4. **Juliet 多材质 + 可拆卸双背带**：棒球手套鞣制皮革 / 光泽皮革 / 麂皮多材质可选，可拆卸斜背带 + 肩带提升背法灵活度。

### 顾客常见问题（FAQ）
**Q1**：这个包是什么皮？
**A1**：光泽皮革，100% 牛皮，在意大利鞣制后再经过数小时滚筒摔软、最后上一层闪亮的光泽釉，所以又有复古做旧感又有耀眼光泽，手感柔软、垂坠度很好。

**Q2**：这是漆皮吗？
**A2**：是光泽皮革(Glazed Leather)，牛皮上釉做出来的光泽效果，和漆皮(Patent Leather)不是一回事；如果喜欢整体光泽造型，可以搭我们的漆皮夹克/短裙做成套。（勿直接说"就是漆皮"）

**Q3**：看起来有点旧/有做旧痕迹，是磨损吗？
**A3**：这是滚筒摔软工艺特意做出来的复古做旧感，是设计风格不是磨损或缺陷。

**Q4**：我平常只买黑色包，这个有合适的颜色吗？
**A4**：光泽皮革新出了很多自然色，特别适合习惯背黑色的人，是很时髦有型的黑色替代，既安全又出彩；也有抢眼的亮色可选。

**Q5**：Juliet 只有这一种皮吗？
**A5**：Juliet 有多种材质款——棒球手套鞣制皮革、光泽皮革、麂皮，还配可拆卸的斜背带和肩带，背法很灵活。

**Q6**：多少钱 / 有多大 / 怎么保养 / 防不防水？
**A6**：以门店主数据为准。（课程未提供光泽皮革的价格、容量尺寸、护理/清洁/防水、退换信息，勿臆测；护理可联动皮革知识课口径或门店 SOP）

### 常见异议及应对
| 关切主题 | 应对方向 |
|----------|----------|
| 做旧是不是显旧/像二手 | 说明做旧复古感是滚筒摔软工艺特意呈现的设计意图，非质量缺陷或磨损 |
| 是不是就是漆皮 | 区分光泽皮革(牛皮上釉)与漆皮(Patent)，不强行等同，可引到光泽造型连带 |
| 习惯买黑色、怕选色踩雷 | 把自然色定位为"时髦的黑色替代"，给既安全又出彩的新选择 |
| 釉面/光泽会不会掉漆、耐不耐用、怎么保养 | 课程未给护理答案——勿编造承诺，以门店主数据/护理 SOP 为准（缺口，见待确认事项） |
