## 概述
Tabby 是 Coach 最具代表性的标志性手袋系列之一，被定位为"现代 Coach 经典手袋"（设计师 Stuart Vevers）。系列以 Soft Grain Leather 为皮革基础，靠"丰富皮革 + 丰富尺寸"覆盖不同顾客；FY26 衍生出 11 个主题子系列（链条 / 纽约 / Y2K丹宁 / 西部 / 超现实等）。属正价店（Retail）明星系列，主要面向质感自用型（手工质量/经典长期主义）与都市悦己型（趋势/自我表达）顾客。价格、库存、精确尺寸课程未提供，以门店主数据为准。

## 产品详细介绍

### 设计灵感与结构
- **系列定位**：Tabby 是最能代表 Coach 品牌的标志性手袋之一，被定义为"现代 Coach 经典手袋"。设计师引用（Stuart Vevers）："Tabby 手袋是现代 Coach 经典手袋，我们要重塑这个包型……"
- **皮革基础——Soft Grain Leather（柔软颗粒纹皮革）**：由经典牛皮制成，纹理柔软、奶油般手感、带天然光泽，每件作品独一无二。
- **品牌叙事**：用时装秀首发 + 权威媒体背书（WWD、Marie Claire）塑造"现代经典 + 趋势引领"的双重叙事，而非单纯讲贵。WWD 引用："Coach 的春季系列是现代风格、轻松、带回忆、手工制作的氛围、风格清新、真诚回归经典。"

### 核心卖点（导购推荐时会用）
1. **现代经典手袋**：Tabby 是最能代表 Coach 的标志性包型之一，"现代 Coach 经典手袋"——主打经典长期主义，不靠"贵/大牌"立论。
2. **丰富皮革 + 丰富尺寸**：从 Soft Grain Leather 到丹宁、羊毛针织、手套鞣皮、光面小牛皮，从小号到 28，每位顾客都能找到适合自己的选择。
3. **每件独一无二**：Soft Grain Leather 经典牛皮天然光泽、每件作品独一无二；报纸印花/拼布等主题强化"独特性 + 手工质量"。
4. **趋势主题叙事**：纽约街区（Times Square）、Y2K 丹宁、西部、超现实扭曲等主题 + 时装秀首发 + WWD/Marie Claire 背书，满足趋势表达与自我风格。

### 客户常见问题（FAQ）
**Q1**：Chain Tabby 能装下我的 iPad / 电脑吗？
**A1**：有三种尺寸——小号可容小型 iPad、中号可容中型 iPad、大号（28）可容 13 寸笔记本，可以按使用场景选（日常通勤选大、轻便晚出选小）。

**Q2**：Knit Soft Tabby（针织款）怎么保养？
**A2**：Knit Soft 是 72% 羊毛 + 聚酯纤维混纺提花织物（Bouclé 纱线），**只能干洗**，不要水洗。

**Q3**：Monochrome Tabby（粉笔色款）需要注意什么？
**A3**：它是"粉笔色时代"全同色系设计，染色烫配的金属配件**容易刮伤**，日常持用时要格外小心。

**Q4**：Soft Tabby、Knit Soft Tabby、Chain Tabby 有什么区别？（易混淆）
**A4**：Soft Tabby 是 2025 春季改款（正面翻盖重新设计、立体 Bombé 凸饰边）；Knit Soft Tabby 是羊毛针织提花款（只能干洗）；Chain Tabby 是链条款（三条可拆卸肩带含链条）。三者材质与护理差异大，别混讲。

**Q5**：Western Tabby 和 Cargo Tabby 是一回事吗？
**A5**：不是。设计团队口径是 Western Tabby"与 Cargo Tabby 类似，再加西部细节"（铆钉、流苏、马鞍切边、工业风插扣口袋）。

**Q6**：这些主题款的灵感是什么？
**A6**：Times Square 取自纽约市（出租车、自由女神、大苹果等）；Quilted Denim 是 Y2K 丹宁趋势；Western 是美式西部；Twisted 取自超现实主义艺术（不对称扭曲）；Sketch/报纸印花取自经典 Peanuts 漫画与 60 年代报纸；Knit Soft 是超大 Signature C 针织图案。

**Q7**：多少钱 / 有没有现货 / 精确尺寸多少 cm？
**A7**：价格、库存、精确尺寸以门店主数据为准【待客户补充】——课程未提供，不臆测。

### 常见异议及应对（写关切主题与方向，不写完整台词）
| 关切主题 | 应对方向 |
|----------|----------|
| 经典款会不会太普通/撞款 | 用主题图案（纽约/西部/Y2K/Peanuts/超现实）+ 工艺细节（刺绣/微章/皮革压印）强调"独一无二"，引导顾客表达自己的审美偏好 |
| 不确定选哪个尺寸 | 回到使用场景选（日常通勤选大、轻便晚出选小）；Chain Tabby 可用 iPad/13寸笔记本作参照 |
| 针织/染色款好不好打理 | 给准护理口径：Knit Soft 只能干洗、Monochrome 金属易刮需小心；按款式分别说明，勿一刀切 |
| 担心是不是真的新款/在被硬推 | 用时装秀首发 + 媒体背书（WWD/Marie Claire）讲"现代经典 + 趋势"，回归服务而非推销姿态 |
| 价格/材质溢价存疑 | 用"现代经典 + 手工质量 + 媒体背书"重构价值，不只强调贵；材质细节交由皮革异议话术（见关联文档） |
