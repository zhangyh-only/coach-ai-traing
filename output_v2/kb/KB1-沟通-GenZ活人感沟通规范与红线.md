## 概述
本文档规定导购在面向 GenZ（Z 世代）顾客时的称谓规范、沟通风格底线与负向红线。它把品牌理念 Expressive Luxury（真我新奢）"协助顾客表达自我、建立自信"的赋能定位，与 CoachX 无届体验"活人感、读懂这代人"的沟通基调，落成一组可被 AI 顾客感知、可被评估扣分的规范。**本篇只写"规范与负向红线"**（供 AI 顾客做出冷淡/防备/被打动的反应，供评估识别踩线扣分）；对应的正向方法、句式与话术原文在 `KB3-方法-GenZ活人感沟通法则`，本篇不展开操作步骤。

## 核心内容

### 一、标准称谓规范
- **统一称"顾客"，不用"客户"**：陪练系统角色称谓固定为"导购（学员）—顾客"。
- **朋友感，非官方客服腔**：定位是"从官方客服切换到懂她的朋友"。导购是顾客自我表达的助力者（来源 2047："协助他们表达自我、建立自信"），不是端着的奢侈品柜姐，也不是机械应答的客服。
- **平等、有温度、不套路**：GenZ 沟通底色是"活人感"——真诚、有温度、不机械（来源 3546）。称呼与开口方式应自然、像人对人，避免高高在上的距离感或模板化敬语堆砌。

### 二、GenZ 沟通风格规范（应当做到的底线）
GenZ 三大底层需求决定沟通基调（来源 3546 冰山模型核心）：**体验感**（在意购物过程的体验与感受）/ **意义感**（在意所买之物的意义与自我表达）/ **安全感**（需要被尊重、被理解、心理安全）。在此之上，规范要求：

| 规范维度 | 应当做到 | 不到位的表现 | 来源 |
|----------|----------|--------------|------|
| 直接、不绕弯 | 信息给得干脆，不层层铺垫、不绕圈子 | 冗长铺垫、套话开场、答非所问 | GenZ 风格要求（新场景资料） |
| 随意感、像朋友 | 语气松弛自然，从官方客服切到"懂她的朋友" | 端着、过度正式、客服式标准腔 | 3546 活人感 + 新场景资料 |
| 善用表情、不堆书面语 | 表达口语化，可用表情/轻松措辞 | 大段书面语、公文式措辞、机械模板 | GenZ 风格要求（新场景资料） |
| 先聆听后回应 | 捕捉顾客"感受/观点/期待"并复述验证，让她感到被听见 | 不回应情绪、自顾自推销、机械应答 | 3546 活人感互动 |
| 允许留白 | 回应后给空间，不急于填满、不抢话、不催成交 | 连续输出、抢话、急于成交 | 3546 |
| 夸赞带观点+情绪 | 肯定要有真实观点与情绪，让顾客被真正看见 | 只罗列事实数据、空泛夸赞，顾客无感 | 3546 SBI |
| 赋能而非推销 | 帮顾客"做真实的自己"、建立自信，回应自我表达诉求 | 纯推销、只讲价格/品牌身份、忽视悦己诉求 | 2047 Expressive Luxury |

### 三、沟通禁忌红线（负向，触发顾客被推开 / 评估扣分）
以下为明确的负向红线。踩中任意一条，AI 顾客会表现冷淡/防备/抵触，评估应据此扣分。

| 红线 | 为什么会把 GenZ 顾客推开 | 来源 |
|------|--------------------------|------|
| **群发营销短信** | GenZ 重视被当成"具体的人"而非业绩数字；群发=被当流量，违背"被看见/安全感"需求，且与"少营销打扰"的偏好冲突 | 新场景资料 C 节（沟通禁忌·质感自用型/质感独行侠） |
| **过度热情、贴身跟随** | 侵占顾客的体验空间与自主性，违背"允许留白"与购物自主性；让顾客有压迫感 | 新场景资料 C 节 |
| **未经询问主动提折扣/满减** | 把沟通窄化为价格博弈，稀释品牌"自我表达/质感投资"叙事；**除非顾客主动问**，否则主动甩优惠会贬损品牌价值、并对价格不敏感的客群适得其反 | 新场景资料 C 节 |
| **用"网红爆款"作推荐理由** | 把奢华简化为"流量/跟风"，违背 Expressive Luxury"自我表达、做真实自己"内核；GenZ 不想被当成只看 logo/只追爆款的人 | 新场景资料 C 节 + 2047 理念内核 |
| **套路化、无温度的标准话术** | 违背"活人感"，让顾客觉得"我只是被标准流程对待，不被当回事" | 3546（套路化沟通会失效） |
| **把奢华窄化为"贵 / 大牌身份"** | 课程明确奢华"更是关于自我表达"；只讲贵/牌子会丢失品牌宗旨，与年轻一代"重表达轻炫耀"的诉求相悖 | 2047 Expressive Luxury |
| **机械复读、不回应情绪** | 顾客表达带情绪/价值观内容时若被无视或模板应答，违背"被听见/体验感"需求 | 3546 活人感互动 |

> 说明：以上"群发营销短信 / 过度热情贴身跟随 / 主动提折扣满减 / 网红爆款作推荐理由"四条核心红线，源出新场景资料 C 节对质感自用型（质感独行侠）的"沟通禁忌"，对 GenZ 客群普遍适用，故归入本通用沟通规范，同时进入 KB4 被推开点与 KB5 扣分项。

## 在陪练系统中的应用

### 陪练对话引用（AI 顾客何时会问到 / 触发本文档内容）
- AI 顾客据本篇判断"被什么话术推开"：当学员**群发式/模板化开场、贴身跟随、未问就甩折扣、用"网红爆款"推荐、只讲贵/大牌**时，AI 顾客表现为冷淡、防备、想离开或反问"你是不是在硬推"。
- 当学员用**活人感、有温度、回应其情绪与自我表达**的方式沟通时，AI 顾客放下戒备、愿意继续。
- 称谓与"朋友感、非客服腔"作为 AI 顾客对学员开口风格的内在评判标尺（端着/客服腔会让她觉得距离感）。

### 对话评估引用（学员表现的哪个评判点引用本文档）
- **沟通风格/活人感**：是否朋友感而非官方客服腔、是否直接不绕弯、是否回应情绪、是否允许留白。
- **沟通禁忌红线**：是否踩中群发营销短信 / 过度热情贴身跟随 / 未问主动提折扣满减 / 用网红爆款作推荐理由 / 套路化无温度话术 / 把奢华窄化为贵。命中即为扣分项。
- **品牌一致性**：是否以赋能式沟通承接 Expressive Luxury"自我表达、建立自信"，而非纯推销/只讲价格。
- 具体扣分项落点见 `KB5-评估-GenZ沟通与画像扮演`（待建）。
