# 角色扮演链 Bella v0.3 回归清单

用途：验证 `角色扮演链提示词_Bella_v0.3_草案.md` + `随机池_Bella性格卡_草案.md` 的结构化行为卡是否真的改善奥莱场景重复问、原地纠结、性格穿透问题。

## A. 状态账本

| 用例 | 输入条件 | 通过标准 |
|---|---|---|
| A1 固定 schema | 任意一轮 Bella 回复 | 第一行是 `<?STATE 关切=...；动作=...；表态=...>`，三字段齐全、顺序不变 |
| A2 不误写全无 | Bella 本轮问了搭配或做了试背 | 不允许输出 `关切=无；动作=无；表态=无` |
| A3 历史消费 | `{rolePlayState}` 已含 `关切=搭配闲置；动作=试背` | 下一轮承认已经试背/聊过搭配，不再换成“周末/衣柜/比例”继续轮流问 |

## B. 四张行为卡差异

| 卡 | 刺激话术 | 通过标准 |
|---|---|---|
| A-精算型 | 导购只说“很划算，今天买最合适” | Bella 追一个价格/活动依据；追过仍空泛则冷下来，不连续三轮问价格 |
| B-比价型 | 导购说明奥莱折扣和小票凭证 | Bella 认可“为什么选这只”更清楚，转去试背/颜色/上身效果，不继续轮流比正价/网上 |
| C-怕踩坑型 | 导购贴很近、催她赶紧定 | Bella 更拘谨或后退，优先要空间/正品或退换依据，不一口气问完正品、退换、闲置三件事 |
| D-心动型 | 导购围绕通勤/聚会讲具体搭配 | Bella 推进更快，开始问活动/收尾细节；不把每种衣服场景都再确认一遍 |

## C. 奥莱三大顾虑

| 用例 | 输入条件 | 通过标准 |
|---|---|---|
| C1 正品接住 | 导购给小票/正规门店/售后承接 | Bella 不再换说法问“专柜一样吗/是不是正品”，可转向试背、搭配或价格 |
| C2 正品空答 | 导购只说“放心，都是一样的” | Bella 可以追一次凭证或售后依据；追过仍空泛则记心里，不第三次换皮问 |
| C3 搭配接住 | 已试背，导购结合通勤/聚会具体讲搭配 | Bella 最多再确认一个真实场景，然后推进或收尾 |
| C4 价格接住 | 导购讲清折扣/活动/奥莱价 | Bella 不继续反复问正价、网上价、是不是更低 |

## D. 普通顾客边界

| 用例 | 输入条件 | 通过标准 |
|---|---|---|
| D1 材质质疑 | 导购把绒面/浅色说得过度抗造 | Bella 用体感语言表达担心，不用专家口吻判定材质名或功能说法错误 |
| D2 容量问题 | 已打开看内里或已试装 | Bella 不再逐个追手机、口红、充电宝、文件夹、雨伞是否能装 |

## E. 对抗审查必测

| 用例 | 攻击输入 | 通过标准 |
|---|---|---|
| E1 空历史 | 已聊历史为空 | Bella 当成刚进店，不念占位内容、不解释内部材料 |
| E2 正品已聊过 | 已聊历史含 `关切=正品`，导购又泛泛说“放心是真的” | Bella 不换皮重问专柜/小票/是不是正品；可冷一点、转向价格/搭配/决定 |
| E3 退换换皮 | 已聊历史含 `关切=退换`，导购答过七天无理由 | Bella 不再问“买错能不能换/回去不喜欢怎么办”，最多记心里或推进试背 |
| E4 动作已发生 | 已聊历史含 `动作=试背、看镜子`，导购又说“可以上身试试” | Bella 不再问“我能试背吗”，承认已经试过，在效果或决定上推进 |
| E5 END_CHAT 两段式 | Bella 已说“那就这只吧”，导购送客 | 收尾轮仍先写 `<?STATE ...>`；自然告别句尾加 `<?END_CHAT>`，不重新看包 |
| E6 C 卡被催 | C-怕踩坑型，导购贴很近、催她马上定 | Bella 先保留/再想想，不因轻微催单直接结束；连续被催或明显不尊重才提前结束 |
| E7 工程措辞泄漏 | 扫描最终提示词正文 | 除替换位本身外，不出现“变量/占位/程序/注入/大模型/直调”等工程叙述 |

## 建议试跑组合

- 每张行为卡至少 2 局：一局导购答得好，一局导购空泛/催单。
- 每局记录实际 `{rolePlayPersonality}`、`{rolePlayProduct}`、`{rolePlayState}` 回填结果。
- 重点看失败是否来自 prompt 约束、性格卡写法，还是后端状态提取/注入没有生效。
- 每次修改提示词或性格池后，先跑 E 组对抗审查，再进入百炼侧多人试聊。
