# 测试诊断：Bella 正品 / 正价差异复发

## 现象

本轮抽到 B-比价型，顾客已经在前半段问过：

- 奥莱店里的货和商场正价柜是不是一样、是不是正品。
- 奥莱官方门店 / 官方系统 / 外面代购私域不对板的区别。

导购也已经答过：

- Coach 奥莱是官方门店，货品是正品。
- 奥莱和正价柜主要差在渠道和产品线，不是真假区别。
- 可以看吊牌、五金、内里、小票，后续售后按门店规则走。

之后 Bella 已经进入试背、看镜子、搭配确认，并多次表态喜欢。但后面又回到：

- “奥莱店里的货跟专柜是一样的吗？”
- “怕买到不对版。”
- “除了价格，做工上真没差？”

## 根因

1. `rolePlayState` 里的 `关切=正品` 被模型读成了“还没解决的待办”，而不是“已经聊过的账本”。提示词虽然写了“已聊历史要认”，但没有明确说“关切出现过就代表已经被提过，出现越多越不能再问”。
2. “正价 / 专柜 / 做工 / 不对版”没有被明确归入同一个底层担忧。模型把“是不是正品”消耗掉后，又把“专柜做工有没有差”当成一个新问题重开。
3. B-比价型性格卡把“奥莱 vs 正价/网上、是不是正品、价格说服力”写得太强，但没有明确要求开局合并问清、后续不要拆成多轮逐项盘。

## 修复

- 新建 `最新使用提示词.md`，不覆盖 v0.3。
- 在 v0.4 中补充：
  - 已聊历史不是待办清单，关切出现过代表已经被提过；同类关切出现越多，后面越不能回头。
  - “专柜差异 / 不对版 / 做工是不是和正价柜一样”都归入正品/渠道安心。
  - “正价差 / 到手价 / 便宜多少”才归入划算价格。
  - 已经问过奥莱和正价/专柜关系，导购给过官方门店/渠道差异/小票/售后依据，且 Bella 已经试背、照镜子、表态喜欢后，正品/渠道线关闭。
- 收紧 B-比价型性格卡：正品/专柜/不对版/做工差异只算同一组，问过并听到官方渠道或售后依据后关掉，不把正价、网上、朋友价、专柜做工轮流问。

## 回归用例

输入条件：

- `{rolePlayPersonality}` = B-比价型。
- `{rolePlayState}` 至少包含两次 `关切=正品`。
- 已聊历史包含 `动作=试背、看镜子`，且 `表态=喜欢`。
- 导购上一轮说“刚才已经说过了啊，都是一样的，都是正品”。

通过标准：

- Bella 不能再问“跟专柜一样吗 / 做工真没差吗 / 是不是不对版 / 正价柜那种会不会更好”。
- 合理出口是冷一点、保留、转价格/收尾，例如：
  - “嗯，我知道你意思了，这个我就不反复问了。那你帮我看下现在到手价吧。”
  - “行，那正品这块我先认了。我主要再纠结下价格和这个颜色会不会太难打理。”
  - “那我再想想吧，主要不是包不好看，是我自己还得消化下预算。”
