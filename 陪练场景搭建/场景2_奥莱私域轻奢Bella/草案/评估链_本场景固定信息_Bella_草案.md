# 本场景固定信息（场景2：奥莱私域轻奢尝鲜族 Bella · 奥莱）

## 顾客画像（固定，判同理心时对照）
- **Bella，26 岁，武汉职场新人**，刚毕业工作两年、月入 6000–8000、正价几千元对她负担偏重。**画像＝性价比入门型（奥莱私域轻奢尝鲜族）**：想用够得着的价格拥有第一只常用 Coach，价格敏感、主动问折扣组合、决策高度依赖闺蜜 / 熟人推荐。
- 她最在意的是「**是不是正品、划不划算、奥莱和正价差在哪、这款能不能经典地背好几年、不喜欢能不能退**」；核心诉求是"正品 + 经典不过时 + 价格只到正价一半"。
- 她**怕踩坑、求安心**：靠闺蜜分享链下单（熟人信任背书、省得自己比价），对"奥莱便宜会不会材质差 / 是不是过季老款"有天然疑虑，但又不好意思直说。
- **判同理心时**：看 SA 有没有读懂她要的是「**划算与安心、而非最贵**」、主动给优惠组合、客观（不贬低）说明 Retail/Outlet 差异、用「经典耐用 / 职场投资」做价格心理按摩、接住「会不会买到假货、会不会买回去后悔」；而不是催买、推超预算高端线、用"奥莱就是次一点"贬低渠道、对渠道疑虑含糊回避。
- 详细潜台词以 **KB4-画像-性价比入门型** 检索为准。

## 渠道（固定）
- **奥莱 Outlet（+ 微信私域触达）**：核心卖点＝"不到正价一半买到正品经典款"。**折扣 / 组合换购是渠道天然属性，SA 主动告知优惠组合算加分、不是红线**（与正价场景相反）；靠**正品保障 + 划算感 + 经典耐用 + 熟人背书**打动。但仍**不靠虚高原价显折扣、不靠贬低正价**；价格 / 折扣 / 活动门槛**以门店主数据为准**。

## 本场景常见事实锚点（核对 SA 时高频用，仍以 KB2 检索为最终依据）
- **正品保障**：奥莱售卖的是品牌正品——SA 应正面、有底气讲清正品与渠道正规（善用熟人 / 私域背书），**不能含糊回避**让顾客更慌。
- **Retail / Outlet 皮革差异**：顾客疑"奥莱便宜是不是材质差 / 过季"时，SA 要**客观说明**差异（生皮 / 分层 / 鞣制 / 修整，见 KB3-方法-皮革材质异议应对话术、课程 3365），**不贬低 Outlet、不顾左右而言他**，不说奥莱是次品 / 假货。
- **Tabby 26**：日常通勤款，**装不下 iPad**（客户 2026-06-25 确认）——SA 说「26 能装 iPad」是硬伤。要装 iPad/电脑看更大尺寸；普通 Tabby 各尺寸（30/35）确切容量 KB2 未明确、以门店主数据为准、不臆造（KB2 仅 **Chain Tabby** 有明确口径：小→小型 iPad、中→中型 iPad、大号 28→13 寸笔记本）。
- **皮质 / 防水**：颗粒纹皮革（「柔软颗粒纹皮革」「抛光鹅卵石纹皮革」是真实表述），别张冠李戴成不存在的名称；防泼水 ≠ 完全防水；天然纹理 / 色差 / 沙眼是**天然属性、非瑕疵**（应归因 +「每只独一无二」，而非含糊承认有问题）。
- **价格 / 折扣 / 组合 / 退换**：奥莱折扣价、换购门槛、私域券**门店主数据未接入，不臆造对错，标「以门店主数据为准」**；退换走"退换无忧 / 未寄出可撤销 / 换转退"，退款时效**不擅自承诺**（以 KB6-服务-交易与服务边界(退换货) 为准、课程 3712）。

## 二、整体评估链额外的「销售五阶段」块

```markdown
## 销售五阶段（评流程意识时对照——本场景到成交为止，不含售后维系）
这场对话理想会走完五个阶段（Bella 不报阶段名，靠她的心理状态 / 反应信号外显）：
1. **迎宾破冰**（CONNECT）：刚进店、有点拘谨 / 精打细算 → SA 给空间不贴身、活人感招呼，不上来就催买。
2. **需求诊断**（CONNECT）：聊想找什么样的包、**有什么活动 / 组合** → SA 开放式问到使用场景 / 职场穿搭 / 预算，主动告知优惠组合，而非上来硬推高价款。
3. **触感体验**（BUILD）：上手摸 / 试背、**顺口问是不是正品 / 奥莱和正价差在哪** → SA 邀请上手试背、材质讲准、**客观说明渠道差异、正面接住正品疑虑**。
4. **价值传递**（BUILD）：权衡**划不划算 / 能背几年值不值** → SA 用价格三明治、BFA 把卖点重构为价值（对 Bella ＝ 划算 + 正品 + 经典耐用 + 职场投资），不夸大权益、不靠虚高原价、不贬低 Outlet。
5. **主动成交**（CLOSE）：释放购买信号（问**正品凭证 / 退换无忧 / 组合换购 / 能不能私域复购**、反复看）、临门犹豫（怕选错 / 想等闺蜜看看） → SA 识别信号、用并行选择法承接犹豫、真诚推一把（划算 + 正品 + 适合），不硬逼；Bella 表态后 SA 顺势送客。
> 查 KB5 / KB3 时用官方四阶段命中更准：CONNECT＝迎宾破冰+需求诊断、BUILD＝触感体验+价值传递、CLOSE＝主动成交。
```
