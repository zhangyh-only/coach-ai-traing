# 实地调研清单 · 真实销售 & 用户情况

> 用途：带去门店，摸清真实销售怎么卖、来的都是些什么人，回来反哺陪练设计。
> 用法：边问边记、能勾就勾。重点是拿到"真实的原话和真实的占比"，别要泛泛的官方说法。

---

## 调研对象 & 怎么开口（先做对这步，后面才聊得到干货）

- **找谁**：① 店长（要数据、要培训资料、引荐人）；② 1–2 个公认能打的老销售（聊实战）；③ 有机会的话，听听老顾客自己怎么说。
- **怎么说**：表明是"来学习、想把你们的经验做成新人培训/陪练工具"，不是来挑刺、不是考核。**销售愿意教，往往是因为这能帮到他们带新人。**
- **挑时机**：别在客流高峰打扰；平峰、午后、或下班后请教更聊得开。
- **要原话**：他说"顾客一般会嫌贵"——追一句"具体会怎么说？""你当时怎么回的？"，**把原话记下来**，这比结论值钱十倍。

---

## 一、问销售：来的都是些什么人

- [ ] 来店顾客大致分哪几种人？（自用 / 送礼 / 比价 / 犹豫 / 老客……）各大概占多少成？　记：________________
- [ ] 哪一类**最多**？哪一类**最难搞**？
- [ ] 男顾客多吗？一般来买啥、什么心态？
- [ ] 有没有那种你一看就知道"今天能成"或"基本悬了"的客人？凭什么判断？

## 二、问销售：顾客怎么拒绝 / 犹豫

- [ ] 顾客**嫌贵**时，最常说的几句原话是什么？　记：________________
- [ ] 说"我再看看 / 再想想"时，她们心里真实在顾虑什么？
- [ ] 有没有线下试好、回头去网上或代购买的？你怎么应对？
- [ ] 最常被问到、又最难答的问题是什么？

## 三、问销售：到底怎么成交的

- [ ] 一单里，哪个动作、哪句话**最决定成交**？
- [ ] 你一般怎么**开场**？怎么判断该不该上前搭话？
- [ ] **嫌贵又不能降价**，你靠什么说服？
- [ ] 怎么让顾客愿意**上手试背**？（很多老销售说这步最关键）
- [ ] 成交后你还会做什么？（搭着多卖一件 / 加微信 / 办会员 / 记喜好）

## 四、问销售：印象最深的客人

- [ ] 最难搞的一次成交是怎样的？最后怎么拿下的？
- [ ] 有没有"怎么都搞不定、只能放弃"的？什么情况？
- [ ] 有没有让你特别有成就感的一单？为什么？

## 五、现场观察（站店里看，别只听）

- [ ] 顾客一进门的真实反应（直奔某只 / 随便逛 / 一脸戒备）
- [ ] 销售在**什么时机**上前、**第一句**说什么、顾客买不买账
- [ ] 哪些话明显有效、哪些一说顾客就冷脸
- [ ] 顾客**试背前后**的态度变化

## 六、要现成材料 / 数据（有就拿，没有就算）

- [ ] 销售培训手册 / 话术手册 / 服务流程 SOP
- [ ] 常见异议和应对的整理（哪怕是口头总结）
- [ ] 客户类型占比、客单价、连带率这些数
- [ ] 能不能录一两段真实接待，或事后口述还原一单

---

## 回来怎么用（带着答案找 Claude 对接）

| 你摸回来的 | 变成陪练里的 |
|---|---|
| 真实客户类型 + 各占多少 | 决定做哪几个虚拟顾客画像、各投放多少 |
| 顾客拒绝/嫌贵的**原话** | 虚拟顾客的"过招"和异议 |
| 成交的**关键动作/话术** | 评分链里"销售做对了"的考核点 |
| 销售怎么开场、怎么留客 | 对话流程每个环节的设计 |

> 一句话：**重点带回"真实的原话 + 真实的占比 + 真实的成交关键动作"**，这三样是我给不了、只能从一线拿的，也是把陪练从"我们编的"变成"贴合真实"的关键。
