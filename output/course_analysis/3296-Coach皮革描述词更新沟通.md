## 课程主题判断

本课程不是新的产品系列课，而是皮革描述词统一口径的内部沟通材料。它应更新皮革与材质知识，而不是生成独立的产品 KB。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 更新时间 | 2025-05-23。 |
| 背景 | 同一种皮革在不同款式中使用不同描述，可能让顾客误解为材质或品质变化。 |
| 更新目的 | 统一产品皮革类型描述，让顾客更清晰理解皮革类型。 |
| 品质口径 | 原材料在任何生产批次中保持一致，品质没有改变。 |
| 例子一 | Brooklyn Shoulder Bag：`NATURAL FULL GRAIN LEATHER` 更新为 `GENUINE FULL GRAIN LEATHER`。 |
| 例子二 | Quilted Tabby：`GENUINE NAPPA LEATHER` 更新为 `GENUINE FULL GRAIN LEATHER`。 |

## 建议回应口径

课程建议导购回应顾客时强调：

- 很开心顾客关注皮革类型描述。
- 本次更新是为了统一不同产品中的用词。
- 原材料始终保持一致，品质没有任何改变。
- 这体现 Coach 对高品质皮革的坚持与专业承诺。
