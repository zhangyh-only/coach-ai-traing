## 课程主题判断

本课程与 2112 同属 E.N.D. 方法域，但重点从“如何使用 E.N.D.”补充到“如何识别购买信号”。因此主结果不是新建第二篇 E.N.D. KB，而是更新同一篇 `KB3-方法-E.N.D.购买信号识别技巧.md`。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 课程目的 | 学习购买信号、为什么重要，以及如何识别这些信号来提升服务。 |
| 信号一 | 持续眼神交流、微笑。 |
| 信号二 | 询问售后服务或 VIP 体系。 |
| 信号三 | 长时间拿着产品、对镜观察。 |
| 信号四 | 询问导购或其他人的意见。 |
| 方法连接 | 识别信号后，可使用 E.N.D. 帮助顾客完成当下购买。 |
