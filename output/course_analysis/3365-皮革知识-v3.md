## 课程主题判断

本课程主讲 Coach 皮革专业知识，包括来源、认证、鞣制、检验、Retail/Outlet 差异、常见顾客疑问和护理建议。其粒度适合形成一篇材质总览型 KB，不按每个皮革类型拆分。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 课程背景 | 区域产品培训团队参加 Dennis Laplume 主讲的 Leather Training，目的是提升门店专业回应能力。 |
| 来源 | Coach 使用天然皮革，主要来自意大利和韩国的 10 家核心制革厂。 |
| 认证 | 皮革来自 LWG 验证的金级或银级制革厂。 |
| 皮革等级 | 使用 A、B 级头层和二层皮革。 |
| 复鞣 | 每种公式为 Coach 自有，用于打造独特皮革。 |
| 检验 | 通过 FFP 与 Flat 测试。 |
| Retail/Outlet 差异 | Retail 多使用头层皮，并可能使用蛇皮、鳄鱼皮等稀有皮革；Outlet 多使用二层皮。 |
| 天然纹理 | 疤痕、生长纹、纹理差异不等同于质量问题。 |
| 色差原因 | 原皮底色、复鞣、染色、喷涂、熨烫、裁切和 QC 都会影响一致性。 |
| 防水 | 水滴停留 5 分钟，自然干燥后不褪色；不等同完全防水。 |
| 存放 | 建议低于 65% 湿度存放。 |

## 常见顾客问题

课程覆盖以下问题：

- 为什么皮革包价格较高。
- Retail 和 Outlet 皮革有什么差异。
- 为什么皮面有很多线条。
- 为什么同一产品纹理不同。
- 为什么羊皮有很多毛孔。
- 为什么光滑皮革容易划伤。
- 沙眼、生长纹是否是瑕疵。
- 什么是头层皮。
- 皮革是否防水。
- 为什么同款包有色差。
