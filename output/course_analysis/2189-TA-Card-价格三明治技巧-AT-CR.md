## 课程主题判断

本课程主讲如何在介绍价格时，把价格置于产品名称与工艺特色之间，使顾客理解产品价值，而不是只关注价格数字。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 方法目的 | 通过渲染产品、分享传统和工艺，让顾客理解价格之外的价值。 |
| 结构一 | 产品名称。 |
| 结构二 | 产品价格。 |
| 结构三 | 产品工艺及特色。 |
| 适用场景 | 刚开始互动、展示产品、推荐加购产品。 |
| 注意事项 | 强调产品价值、特色和优势。 |

## 讨论问题

- 如何根据顾客生活方式和需求使用价格三明治来触发购买欲望。
- 如何让顾客关注价值，而不是只关注价格。
