## 课程主题判断

本课程主讲顾客提出异议时的处理方法。核心不是直接反驳，而是让顾客感到安心、被听见，并相信导购在寻找最适合其需求的产品。

## 抽取事实

| 模块 | 事实 |
|------|------|
| Acknowledge | 回应顾客顾虑。 |
| Ask | 询问，进一步理解异议原因。 |
| Address | 处理解决，给出有效方案。 |
| 关键提醒 | 理解顾客未说出口的异议。 |
| 风险提醒 | 不预判顾客想法，不打断或立即纠正，不缺少开放式问题，不给无效方案。 |

## 异议示例归纳

| 原始表达 | 归纳主题 |
|----------|----------|
| 这不是我心目中想象的红色。 | 颜色预期不符。 |
| 你确定我穿这个好看吗？ | 上身效果和自信不足。 |
| 如果我以后讨厌这个产品怎么办？ | 购买后悔风险。 |
| 你是不是只想要把这个东西卖掉才鼓励我买？ | 对导购动机的信任疑虑。 |
