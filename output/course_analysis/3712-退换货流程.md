## 课程主题判断

本课程是售后/退换货流程操作资料，面向门店处理和顾客说明。主价值在于让导购知道不同节点的处理动作和边界，因此归入 KB1 交易与服务边界。

## 抽取事实

| 流程节点 | 课程事实 |
|----------|----------|
| 撤回售后申请 | 退换货顾客未邮寄退回商品前，可根据实际情况撤销售后并关闭售后单。 |
| 售后处理流程 | 店长审批通过的售后申请，系统自动创建售后单；门店售后找到该售后单。 |
| 返回门店/返回仓库 | 售后申请选择返回门店时，退货地址显示门店地址；选择返回仓库时，显示仓库收货地址。 |
| 返回仓库建议 | 为减少质检问题，自营店申请退货时建议优先退回仓库。 |
| 顾客寄回 | 店员将退货地址转发给顾客，与顾客沟通寄回退货商品，并让顾客提供寄回物流公司和单号。 |
| 填写物流 | 顾客寄回后，将顾客提供的物流公司和单号填写到售后单中；如果列表中不存在对应物流公司，可选择“其他”自行填写。 |
| 待退货商品寄回 | 售后单进入待退货状态，等待退货商品寄回。 |
| 退货入库 | 商品已退回门店后，可切换到退货履约菜单，通过扫描退货包裹快递单号快速查找退货单。 |
| 包裹核验 | 点击退货入库后，打开包裹物流单号核验弹窗；通过退货包裹快递单号条码再次核验包裹是否与当前退货单一致。 |
| 商品验收 | 包裹核验一致后，打开包裹逐一检查退货商品，并扫码商品条码进行确认验收；若商品标签缺失需使用自定义条码完成退货。 |
| SA 检查提醒 | 课程强调 SA 检查时尽量拍摄视频确认无误再点击下一步。 |
| 入库完成与退款 | 退货商品验收合格完成后，点击确认入库完成退货；退货入库完成后，自收银门店售后单将自动退款，第三方收银门店需要手动确认完成退款。 |
| 换转退 | 申请换货后，在退回商品已入库、换货商品发货前，可根据顾客需求取消换货并申请退款；店长审批后进入退款。 |
| 第三方收银门店退款确认 | 第三方收银指非公司财务账号自收银以外的第三方收银，如联营商场；在第三方 POS 退款后，Portal 上对应售后单需手动确认完成退款。 |
