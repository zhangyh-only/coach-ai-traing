## 抽取事实

| 模块 | 事实 |
|------|------|
| 系列定位 | Tabby 是 Coach 现代经典和明星产品系列之一。 |
| 设计引用 | Stuart Vevers 表示 Tabby 是现代 Coach 经典手袋。 |
| 总体卖点 | 不同皮革、精致设计和丰富尺寸，让顾客找到适合自己的选择。 |
| Peanuts / 报纸印花 | 使用经典漫画和报纸印花主题。 |
| Sketch Print | 通过刺绣、微型吊饰和皮革压印表现涂鸦印花。 |
| Chain Tabby | 2025 春季时装秀亮相，Soft Grain Leather，立体 C 金属配件，磁扣闭合，三条可拆卸肩带。 |
| Chain Tabby 尺寸 | 小号可容纳小型 iPad；中号可容纳中型 iPad；大号可容纳 13 寸笔记本电脑。 |
| Times Square | 纽约主题，灵感来自黄色出租车、I Heart New York 马克杯、自由女神像、大苹果等。 |
| Quilted Denim | 更年轻、休闲、随性。 |
| Patchwork | 旧丹宁面料拼贴，体现可持续和减少环境破坏的承诺。 |
| Western | 西部风格，含铆钉、流苏、毛毡牛仔帽、马鞍风格切边等灵感。 |
| Shoulder Bag 28 | 小巧廓形，精致皮革、金属配件和水晶，适合晚宴或约会造型。 |
| Twisted Tabby | 不对称包型，C 金属配件带弯曲和扭曲外观。 |
| Soft Tabby | 正面翻盖重新设计，搭配 Chain Tabby 伸展台方向。 |
| Monochrome | 粉笔白同色系，缝线、金属配件和皮牌均为粉笔白；染色金属配件易刮伤，需提醒小心。 |
| Knit Soft Tabby | 使用 72% 羊毛和聚酯纤维混纺提花织物，Bouclé 纱线，护理指南为只能干洗。 |
