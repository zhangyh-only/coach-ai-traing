## 课程事实提取

### 课程定位

课程说明：学习者已经了解“我们的宗旨以及驱动我们的核心价值观”之后，本课程继续说明作为 Coach 员工应如何体现这些核心价值，并在店铺内落实这些核心价值，以帮助个人和团队培养良好的工作文化。

### 核心价值 1：真我同行

课程原文要点：

- “我们知道，当不同想法与观念相互激荡冲撞时，绝对能缔造卓越非凡的成果！”
- Coach “推崇个人特质、鼓励大家在各个层面发表自己的想法”。

店铺实践方式：

| 实践方式 | 原文依据 |
|----------|----------|
| 自我表达 | 在团队会议以及其他场合，让大家听见你的声音。 |
| 与队友合作 | 一起集思广益，想出最棒的策略来推动业务。 |
| 开发人际网络 | 与其他店铺合作，让了不起的事情能够成真，例如分享最佳实践、举办联动活动等。 |
| 营造包容环境 | 领导者应营造包容接纳的环境，让团队成员分享新的想法。 |

补充资源线索：

- 员工资源小组 ERG。
- Loop 中的 EI&D Central。
- Coach Foundation：属于 Coach 的组织，重点在于扶持以社群为协助对象的全球慈善倡议，目的是替所有人创造更美好的未来。

### 核心价值 2：探索求新

课程原文要点：

- “你的想法对我们来说很重要！”
- Coach 鼓励大家“透过想出新的、有创意的工作方式，来对店铺业务有所贡献”。

店铺实践方式：

| 实践方式 | 原文依据 |
|----------|----------|
| 分享你的想法 | 不要害怕与领导者沟通对话来获得反馈。没有什么想法是不能表达的。 |
| 测试与学习 | 经过店铺经理同意之后，尝试一些新的东西，并且与团队分享成果。 |
| 领导者沟通 | 领导者应腾出时间来与团队沟通，可以一对一，也可以同时和整个团队沟通。 |

### 核心价值 3：坚守本心

课程原文要点：

- “每个人都应该有权利在工作中做自己！”
- 学员应“扮演一个积极主动的角色，来创造一个开放包容的环境，并以积极正向和相互尊重为基础来打造这个环境”。

店铺实践方式：

| 实践方式 | 原文依据 |
|----------|----------|
| 诚实坦白 | 诚实坦白，敞开胸怀来沟通。 |
| 积极倾听 | 当别人与你分享想法时，要专心、积极倾听。 |
| 保持积极开放 | 永远都要预设大家是保持良善的意图在跟你互动。 |
| 领导者以身作则 | 当领导者有勇气表现出真实的自我时，也会赋予团队成员同样的力量。 |

### 核心价值 4：匠心传承

课程原文要点：

- “我们全心全意细心打造产品，但更重要的是，我们也全心全意栽培员工！”
- Coach 希望员工成长、茁壮，并与 Coach 一起发展长远的职业生涯。

店铺实践方式：

| 实践方式 | 原文依据 |
|----------|----------|
| 分享职业目标 | 与店铺领导者分享职业目标，有助于和店铺经理制定业务计划，让员工达成目标。 |
| 建立跨店导师 / 学员关系 | 建立关系后，能持续获得最新信息与业务近况。 |
| 探索教育机会 | Coach 有丰富的资源与计划，员工可在职业生涯过程中利用，例如专业培训计划、WeLearning 学习平台、蜂行计划、实习等。 |
| 关注新成员目标 | 团队有新成员加入时，领导者应询问他们在 Coach 的职业生涯目标；查看工作状况时持续对话互动，看看能提供哪些协助。 |
