## 课程主题判断

本课程主讲 E.N.D. 如何帮助顾客在当下购买产品。课程重点是导购在顾客犹豫时，使用专业、坚定、被理解的方式帮助其做决定。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 使用场景 | 顾客已经准备购买，但仍有犹豫或无法决定。 |
| 关键态度 | 态度坚定、展现专业，同时让顾客感觉被听见和理解。 |
| E - Eliminate | 排除选项或犹豫点，缩小选择范围。 |
| N - Needs | 重申顾客分享的功能性需求和情感性需求，并说明目标产品如何满足。 |
| D - Decision | 用强而有力的陈述帮助顾客决定，并自信邀请购买。 |
| 团队销售 | 同事在旁且有空时，可纳入对话帮助顾客做决定。 |

## 课程示例

课程示例围绕 Backpack 展开：

- 排除黄色款，因为顾客想每天背，黄色更抢眼、不如海军蓝百搭。
- 重申顾客不想再买黑色手袋，海军蓝能给穿搭注入低调色彩，并可放入 15 寸笔记本电脑。
- 最后以专业信心确认这款能满足需求、背起来好看，并邀请帮顾客包起来。
