## 课程主题判断

本课程不是单一产品课，而是通过真实门店案例让导购学习如何识别 GenZ 顾客购买第一只 Coach 的纪念时刻，并把成长节点转化为服务和成交机会。

## 抽取事实

| 主题 | 课程事实 |
|------|----------|
| 核心洞察 | GenZ 顾客不只是为了“买一个产品”走进门店，他们常在成长节点、身份变化和重要关系中购买第一只 Coach。 |
| 项目形式 | RT&D 团队将门店真实故事制作成系列视频，分三期不同主题，每期推送两支影片给全员。 |
| 三期主题 | `关于成长的 第一次`、`关于爱与自我表达`、`关于告别 与 新开始`。 |
| 成长节点 | 奖学金、实习、毕业、第一次工作、特定亲人的礼赠等都可能成为第一只 Coach 的购买动机。 |
| 学习目的 | 学习成功案例、发现店铺机会点、借鉴并行动，把“纪念时刻”变成服务日常。 |
| 第一主题 | 第一主题聚焦学生时代、自我认同，以及“被看见”“纪念一份努力”等动机。 |
| 后续共创 | 若门店也遇到“第一只 Coach”故事，可反馈给 RT&D，真实故事可能成为下一位主角。 |
