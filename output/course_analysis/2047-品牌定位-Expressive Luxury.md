## 抽取事实

| 类型 | 事实 |
|------|------|
| 品牌定位 | `Expressive Luxury` 是 Coach 对奢华持续演进的想象。 |
| 核心含义 | 这种奢华不只是透过品牌表达价值与概念，更是关于自我表达的精神，即拥抱各种面向的自己。 |
| 代际背景 | 奢华的角色正在变化，并由年轻一代驱动；他们正在重塑自我表达方式。 |
| 品牌宗旨 | Coach 的品牌宗旨是激励顾客自信探索自己的各个面向，并拥有真实做自己的勇气。 |
| 店铺实践 | 通过 Coach 顾客体验，以产品与服务协助顾客表达自我、建立自信。 |
