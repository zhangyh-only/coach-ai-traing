## 课程主题判断

本课程主讲 Gen-Z 沟通技能，重点是管理者如何用“活人感”方式与 Gen-Z 建立连接，并进行正向或改进型反馈。课程不主讲产品、销售流程或顾客画像，因此不拆分到 KB2/KB3/KB4。

## 抽取事实

| 模块 | 事实 |
|------|------|
| 活人感沟通 | 分为聆听和回应两步。 |
| 聆听 | 捕捉冰山下的感受、观点、期待。 |
| 回应 | 验证听到的感受 / 观点 / 期待是否准确，并允许留白、适时提问。 |
| SBI 正向反馈 | Situation 情景、Behavior 行为、Impact 影响。 |
| SBI 使用原则 | 情景可简短或省略；肯定要带观点；关注行为背后的努力，并强化具体正向行为。 |
| SBIR 改进型反馈 | 通过 B 问 B 答澄清原因、共创行动，并注入信心。 |
