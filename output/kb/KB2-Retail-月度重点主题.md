### 3. 主题推荐方式

| 顾客表达 | 推荐处理 |
|----------|----------|
| “我想要夏天一点的包。” | 从 Straw Shop、木质串珠、草编和高饱和色调切入。 |
| “我想要有质感，但不要太高调。” | 可引导深巧克力色麂皮、条纹拼接皮革或较低调的 Signature 帆布。 |
| “我想要拍照好看。” | 先确认拍照场景，再选择草编、串珠、流苏、高饱和色调等视觉更明显的主题。 |
| “我怕太季节性。” | 引导到材质、颜色和日常搭配频率，不用趋势压过实用需求。 |

### 4. 常见异议及应对

| 关切主题 | 应对方向 |
|----------|----------|
| “草编/串珠会不会只适合度假？” | 可以作为造型亮点，也可搭配日常基础色服装；但不承诺所有场景都适合。 |
| “高饱和色会不会难搭？” | 让顾客用现有衣橱颜色来判断，先从小面积或中性色穿搭中做亮点。 |
| “麂皮会不会难打理？” | 回到材质护理边界，建议后续补充护理资料，不做防水耐污承诺。 |
| “Signature 会不会太明显？” | 区分漆皮 Signature、条纹 Signature 帆布等不同视觉强度。 |
