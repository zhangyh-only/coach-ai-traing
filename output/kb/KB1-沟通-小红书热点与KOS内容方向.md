## 概述

本文档记录 Coach KOS 小红书热点快报中的内容方向和表达边界。它主要服务于 GenZ 语境下的内容种草、私域沟通和门店伙伴社媒表达，不直接替代到店销售话术。

## 核心内容

| 维度 | 内容 |

### 2. 可确认热点方向

| 话题方向 | 可迁移到 Coach 的表达方式 |
|----------|----------------------------|
| `#ColorWalk` | 用色彩作为内容入口，把包袋、穿搭和城市行走场景连接起来。 |
| `#OHduty` | 围绕通勤、下班后、日常切换等场景表达产品陪伴感。 |
| `#SoftThings` | 用柔软、治愈、舒适等情绪关键词描述材质和生活状态，但不夸大材质性能。 |
| `Sportique` | 将运动休闲、轻松出行和年轻穿搭结合，适合表达不费力的时髦感。 |
| `Staycation` | 把周边度假、短途出游、城市松弛感与包袋搭配结合。 |

### 3. 内容表达原则

| 推荐做法 | 避免做法 |
|----------|----------|
| 先讲生活场景，再自然带出产品。 | 只堆商品图、型号和硬卖点。 |
| 用真实穿搭、出行、通勤、周末计划承接产品。 | 把热点标签生硬贴到完全无关的商品上。 |
| 语气轻松、像真实分享，保留个人视角。 | 使用过度官方、过度营销的品牌宣言。 |
| 结合颜色、材质、包型和使用场景做内容。 | 夸大产品效果、服务承诺或未确认权益。 |
| 让图片和文案共同呈现场景。 | 只靠标题跟风，没有内容支撑。 |

### 4. 陪练中可用的顾客反应

AI 顾客可以表现出以下社媒驱动的购买心理：

- “我在小红书刷到这个颜色，感觉挺适合春夏。”
- “我想要一个拍照好看、但平时也能背的包。”
- “这款是不是比较适合周末出去玩？通勤会不会太夸张？”
- “我不想买一眼广告感很强的东西，想要自然一点。”

导购需要把社媒话题转回顾客真实使用场景，而不是只说“现在很火”。
