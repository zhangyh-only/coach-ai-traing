## 概述

本文档记录 Coach 对 `Expressive Luxury`（真我新奢）的品牌定位、品牌宗旨，以及员工在店铺中践行品牌核心价值的原则。`2047_品牌定位： Expressive Luxury` 补齐了真我新奢和品牌宗旨的正式定义；`2293_在店铺中践行品牌宗旨` 补充了四项核心价值在店铺文化、团队协作和领导行为中的落地方式。

## 核心内容

### 1. Expressive Luxury / 真我新奢的定义

课程将 `Expressive Luxury` 表述为 Coach 对奢华持续演进的想象。它不只是透过品牌表达一种价值与概念，更强调自我表达的精神，也就是拥抱各种面向的自己。

在陪练中引用时，应把 `Expressive Luxury` 理解为 Coach 鼓励顾客表达真实自我、建立自信的品牌定位，而不是单纯的价格、稀缺性或传统奢华等级。

### 2. 品牌宗旨

Coach 的品牌宗旨是激励顾客自信探索自己的各个面向，并拥有真实做自己的勇气。

课程强调，团队每位成员在实践这项使命时都很重要。通过真诚、勇敢的自我表达，员工可以激励他人采取同样行动，并发挥正面影响力。

### 3. 在店铺中落实真我新奢

课程给出的落地方式是通过 Coach 顾客体验，以合适的互动方式、产品和服务协助顾客表达自我、建立自信。

可用于陪练的行为方向：

| 行为方向 | 陪练引用方式 |
|----------|--------------|
| 鼓励真实表达 | 导购应尊重顾客的个人风格、偏好和自我表达，而不是只按单一审美判断。 |
| 连接产品与自信 | 介绍产品时要说明它如何帮助顾客表达自己，而不是只堆叠材质和工艺点。 |
| 以体验承接品牌定位 | 服务过程应让顾客感觉被理解、被支持，并有信心做出选择。 |
| 不把品牌理念讲成口号 | 陪练中可把品牌宗旨作为导购行为底层约束，不要求导购机械复述全文。 |

### 4. 核心价值一：真我同行

课程强调，不同想法与观念相互激荡时，可以缔造卓越成果。Coach 推崇个人特质，鼓励团队在各个层面表达想法。

| 店铺实践原则 | 课程依据 |
|--------------|----------|
| 自我表达 | 在团队会议以及其他场合，让大家听见你的声音。 |
| 与队友合作 | 一起集思广益，想出推动业务的策略。 |
| 开发人际网络 | 与其他店铺合作，分享最佳实践、举办联动活动等。 |
| 营造包容环境 | 店铺领导者应营造包容接纳的环境，让团队成员分享新想法。 |

### 5. 核心价值二：探索求新

课程强调，员工的想法对 Coach 很重要。Coach 鼓励员工通过新的、有创意的工作方式，对店铺业务有所贡献。

| 店铺实践原则 | 课程依据 |
|--------------|----------|
| 分享想法 | 不要害怕与领导者沟通并获得反馈。 |
| 测试与学习 | 经过店铺经理同意后尝试新做法，并与团队分享成果。 |
| 主动沟通 | 店铺领导者应腾出时间与团队沟通，可一对一，也可面向整个团队。 |

### 6. 核心价值三：坚守本心

课程强调，每个人都应该有权利在工作中做自己。员工应积极主动地创造开放包容的环境，并以积极正向和相互尊重为基础。

| 店铺实践原则 | 课程依据 |
|--------------|----------|
| 诚实坦白 | 诚实坦白，敞开胸怀来沟通。 |
| 积极倾听 | 当别人分享想法时，要专心、积极倾听。 |
| 保持积极开放 | 预设大家是保持良善意图在互动。 |
| 以身作则 | 领导者表现真实自我，也会赋予团队成员同样力量。 |

### 7. 核心价值四：匠心传承

课程强调，Coach 全心全意细心打造产品，也全心全意栽培员工，希望员工成长、茁壮，并与 Coach 一起发展长远职业生涯。

| 店铺实践原则 | 课程依据 |
|--------------|----------|
| 分享职业目标 | 与店铺领导者分享职业目标，有助于制定业务计划。 |
| 建立跨店导师 / 学员关系 | 帮助员工持续获得最新信息与业务近况。 |
| 探索教育机会 | 可利用专业培训计划、WeLearning 学习平台、蜂行计划、实习等资源。 |
| 持续关注新成员 | 领导者应询问新成员在 Coach 的职业目标，并持续提供协助。 |

## 在陪练系统中的应用

### 对话采集与变量生成

- 当场景涉及品牌定位、价值传递、风格表达或顾客自信时，可检索本文档补全品牌背景。
- 当场景需要判断导购是否理解 Coach 的品牌宗旨时，可引用 `Expressive Luxury` 与品牌宗旨，而不是只引用内部核心价值。
- 当场景涉及团队文化或店铺领导力时，可引用四项核心价值；面向顾客的销售陪练中，应避免把内部文化内容直接讲给顾客。

### 陪练对话引用

- AI 顾客可围绕个人风格、自我表达、是否适合自己等问题展开，而导购应把产品价值连接到“帮助顾客表达自我、建立自信”。
- 导购可以自然表达 Coach 鼓励真实自我的品牌态度，但不应把品牌宗旨当作背诵话术。
- 当顾客对风格选择犹豫时，导购应通过理解需求、尊重偏好和支持决策来体现品牌定位。

### 对话评估引用

| 评估点 | 对应课程依据 |
|--------|--------------|
| 是否能把产品介绍连接到顾客自我表达 | Expressive Luxury / 真我新奢 |
| 是否理解 Coach 品牌宗旨 | 激励顾客自信探索自己的各个面向，拥有真实做自己的勇气 |
| 是否尊重不同想法与个体特质 | 真我同行 |
| 是否鼓励创新和测试学习 | 探索求新 |
| 是否诚实坦白、积极倾听、预设善意 | 坚守本心 |
| 是否理解 Coach 对员工成长和长期发展的重视 | 匠心传承 |
