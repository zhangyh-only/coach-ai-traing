## 概述

本文档记录 `Glazed Leather Juliet` 课程中的 Juliet Shoulder Bag 和光泽皮革相关卖点。课程重点是用全新光泽皮革、自然色调、可拆卸背带和 Juliet 的结构细节，帮助导购向顾客介绍柔软垂坠、复古做旧和街头穿搭灵感。

## 核心内容

### 1. 相关产品

| 产品 | 课程信息 |
|------|----------|
| CR652 Glazed Leather Juliet Shoulder Bag | 光泽皮革 Juliet Shoulder Bag，是课程主讲产品。 |
| CR651 Glazed Leather North Tote 32 / Nomad Tote 32 | 同样用于展示光泽皮革的产品线索。 |
| Coated Canvas Signature Mini Crescent Bag CR510 | 课程页中作为相关搭配或同页产品出现。 |
| CS423 Patent Leather Jacket、CS489 Patent Leather Skirt | 当顾客喜欢光泽造型时，可展示漆皮成衣作为风格延伸。 |

### 2. 光泽皮革卖点

| 卖点 | 课程事实 |
|------|----------|
| 产地和材质 | 光泽皮革来自意大利，课程写明为 100% 牛皮。 |
| 触感 | 具有独特光泽和柔软垂坠感。 |
| 工艺 | 通过自然滚筒摔软工艺形成复古做旧感，再加入闪亮光泽。 |
| 视觉效果 | 颜色多样，天然色调可作为黑色手袋之外的选择。 |
| 风格定位 | 适合追求时髦、街头穿搭灵感和光泽造型的顾客。 |

### 3. Juliet 重点特色

| 特色 | 说明 |
|------|------|
| 皮革版本 | 课程提到棒球手套鞣制皮革、光泽皮革、麂皮等 Juliet 材质。 |
| 设计呼应 | 设计呼应 Lana 与 Nomad。 |
| 品牌元素 | 配有立体雕塑感 C 标识。 |
| 背带结构 | 可拆卸斜挎背带和肩背带，并通过夹扣固定。 |
| 颜色和材质选择 | 有自然色调、大胆色彩和不同材质版本。 |

### 4. 推荐话术方向

- 对通常选择黑色手袋的顾客，可推荐天然色调作为更柔和但仍百搭的替代选择。
- 对喜欢街头感、光泽感、复古做旧质感的顾客，可强调光泽皮革的意大利来源、柔软垂坠和滚筒摔软工艺。
- 对重视搭配灵活性的顾客，可强调可拆卸肩背带和斜挎背带。
- 对喜欢光泽造型的顾客，可延伸展示漆皮成衣，但不把成衣作为 Juliet 的必备组合。
