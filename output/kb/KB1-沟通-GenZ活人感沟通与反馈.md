## 概述

本文档记录 Coach Gen-Z 精神指南课程中的“活人感沟通”、正向反馈和改进型反馈方法。课程原始对象主要是管理者与 Gen-Z 员工的沟通；在陪练中可迁移为 GenZ 顾客沟通原则，但不应把员工管理场景的反馈话术直接当成顾客销售话术。

## 核心内容

### 1. Gen-Z 冰山模型

课程使用冰山模型提醒管理者：可见行为只是表层，真正影响 Gen-Z 反应的是冰山下的感受、观点和期待。

| 层级 | 课程表达 | 陪练引用方式 |
|------|----------|--------------|
| 表层 | 行为、表面的信息 | 顾客说“再看看”“不确定”时，不只看表面拒绝。 |
| 冰山下 | 感受、观点、期待 | 追问她在意什么、担心什么、期待什么体验。 |
| 环境因素 | 成长环境、职场行为、核心观念 | 用于理解 GenZ 更看重参与感、意义感、安全感和真实表达。 |

课程页中出现的 Gen-Z 冰山关键词包括：数字原住民、全球化大融合、5G 冲浪体验官、真实勇敢做自己、情绪自由我做主，以及体验感、意义感、安全感。

### 2. 活人感沟通

课程总结：以好奇心代替偏见，是与 Gen-Z 有效沟通的前提。活人感沟通分两步：

| 步骤 | 核心动作 | 说明 |
|------|----------|------|
| 聆听 | 捕捉冰山下的感受、观点、期待 | 不急着判断，也不急着纠正，先听出对方真正关心的东西。 |
| 回应 | 验证听到的感受、观点、期待是否准确 | 用回应让对方感到被听见，并允许留白、适时提问。 |

可使用的回应句式：

- 我能感受到你……
- 我能感受到你认为 / 觉得……
- 我能感受到你期待 / 希望 / 想要……

### 3. 有效反馈三要素

课程将反馈拆为三个要素：

| 要素 | 说明 |
|------|------|
| 具体的 | 反馈要指向具体情境和具体行为。 |
| 及时的 | 不要拖到对方已经失去情境记忆时才反馈。 |
| 平衡的 | 既关注事实，也关注影响和改进方向。 |

课程结论是：只有当 Gen-Z 被听见时，他们才会听你说的。

### 4. 正向反馈：SBI 夸夸技巧

SBI 用于正向反馈。

| 字母 | 课程含义 | 使用原则 |
|------|----------|----------|
| S | Situation 情景 | 双方都知晓的情景可简短或省略。 |
| B | Behavior 行为 | 关注具体正向行为，并看见行为背后的努力。 |
| I | Impact 影响 | 不只陈述事实，还要带有观点的肯定。 |

课程提醒，“这个月 Tabby 卖得很好”只停留在事实；更有效的夸夸要表达观点，并追问对方是怎么做到的，从而强化具体正向行为。

### 5. 改进型反馈：SBIR 的 B 问 B 答

课程使用“B 问 B 答”讲改进型反馈：先描述情境、行为和影响，再通过提问澄清并共创行动，而不是直接批评。

| 动作 | 说明 |
|------|------|
| 提问澄清 | 询问“是不是遇到了什么事情”，理解行为背后的原因。 |
| 提问共创 | 询问“如果以后再遇到类似情况，我们怎么处理比较好”。 |
| 注入信心 | 表达“我完全相信你肯定可以做好”这类支持感。 |
| 避免指责 | 不用命令式、贴标签式、羞辱式反馈。 |

### 6. 沟通风格边界

| 推荐做法 | 避免做法 |
|----------|----------|
| 非模板化、真实表达、真诚倾听。 | 套话、机械照本宣科、端着说教。 |
| 以好奇心理解对方的观点和期待。 | 先入为主地评价“年轻人就是怎样”。 |
| 反馈具体、及时、带观点。 | 只说“不错”“加油”或只列事实。 |
| 给对方留白和回应空间。 | 立刻纠正、打断、压过对方表达。 |

## 在陪练系统中的应用

### 陪练对话引用

- 当 AI 顾客是 GenZ 或表达强烈个人风格时，可引用本文档中的“先听感受、观点、期待，再回应”的沟通原则。
- 当顾客提出“我不确定”“感觉不太像我”“我想再看看”时，导购应避免立刻推销，先识别冰山下的感受与期待。
- 导购语言应更真实、有朋友感，避免过度正式、模板化和高压式沟通。

### 对话评估引用

| 评估点 | 合格表现 |
|--------|----------|
| 倾听深度 | 能识别顾客表面表达背后的感受、观点和期待。 |
| 回应方式 | 能验证理解是否准确，而不是直接下判断。 |
| GenZ 风格 | 表达自然、真诚、非模板化。 |
| 反馈能力 | 在内部培训或管理类陪练中，可检查 SBI / SBIR 是否具体、及时、平衡。 |
