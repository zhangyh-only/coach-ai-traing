## 概述

本文档汇总 Coach 皮革来源、加工、检验、Retail 与 Outlet 材质差异、常见顾客疑问，以及 2025 年皮革类型描述更新口径。它用于支持导购在面对皮革品质、纹理差异、划痕、防水、颜色差异和描述词变更等问题时，给出一致、专业且不夸大的解释。

## 核心内容

### 1. Coach 皮革来源与标准

| 主题 | 课程事实 |
|------|----------|
| 原料来源 | Coach 使用天然皮革，主要来自意大利和韩国的 10 家核心制革厂。 |
| 环境标准 | Coach 选择经过 LWG 验证的金级或银级制革厂。 |
| LWG 含义 | LWG 即 Leather Working Group，用于评估环境、社会责任和供应链合规表现。 |
| 皮革等级 | Coach 使用 A、B 级头层和二层皮革。 |
| 复鞣 | 每种复鞣配方均为 Coach 自有配方，用于打造独特 Coach 皮革。 |
| 检验 | 皮革需通过 FFP 与 Flat 测试。 |

### 2. Retail 与 Outlet 皮革差异

| 维度 | Retail | Outlet |
|------|--------|--------|
| 原料范围 | 除牛皮、羊皮外，还可能使用蛇皮、鳄鱼皮等稀有皮革。 | 主要使用牛皮和羊皮。 |
| 皮层使用 | 主要使用头层皮。 | 多使用二层皮。 |
| 鞣制取向 | 工艺相对少，更强调天然纹理。 | 课程未展开具体流程。 |
| 表面效果 | 可见蜡感、丝滑感、半植鞣等效果。 | 课程未展开具体效果。 |
| Signature C 帆布 | C 字母更小，深底浅字，其中一个 C 中隐藏 Coach 标识。 | 浅底深字，无额外隐藏签名。 |

### 3. 天然纹理不是质量问题

Coach 使用天然牛皮，皮面可能出现疤痕、生长纹、色差和纹理差异。课程强调，这类天然痕迹不是质量问题。设计和裁切会尽量保留天然质感，同时通过隐藏部位使用等方式减少浪费，体现可持续理念。

可回答的顾客疑问：

- 为什么皮面有纹路或小痕迹？
- 为什么同款产品的纹理不完全一致？
- 为什么羊皮有明显毛孔？
- 为什么光滑皮革容易留下划痕？

### 4. 同款纹理和颜色为什么不同

| 原因 | 说明 |
|------|------|
| 天然原料差异 | 每张皮革天然颜色、纹理和底色都有差异。 |
| 后整理工艺 | 喷涂、熨烫、压花、滚筒摔软等工艺会形成独特纹理。 |
| 复鞣和染色 | 课程提到复鞣约 12 小时，再进行染色，影响颜色渗透和最终表现。 |
| 裁切标准 | 同一只包的部片会尽量来自同一张皮；同一款不同包之间仍可能存在天然差异。 |
| 质量检验 | 通过 FFP、Flat、颜色标准和色牢度等测试确保品质。 |

### 5. 头层皮与二层皮

湿皮具备一定厚度时，紧实的上层被切出后即为头层皮，用于头层牛皮制品。其他分层可用于仿麂皮或带 PU 封层和全纹压花的涂层皮革。课程强调，头层和二层皮都需要通过 Coach 的 FFP 要求。

### 6. 防水和保养

| 主题 | 课程事实 |
|------|----------|
| 防水测试 | 水滴在皮革表面停留 5 分钟，自然干燥后不褪色。 |
| 保护膜 | Coach 通过独特喷涂后整理工艺和专属颜料形成保护膜。 |
| 受潮处理 | 若皮革沾水，应尽快用无绒干毛巾处理，并置于阴凉通风处自然干燥。 |
| 门店服务 | 顾客可将包带到 Coach 店铺进行皮革清洁和保养服务；课程提到保养流程中可能包含防水喷雾。 |
| 存放湿度 | 课程建议低于 65% 湿度存放。 |

### 7. 皮革描述词更新口径

2025 年皮革类型描述更新的目的，是统一不同产品中的皮革类型描述，让顾客更清晰地理解所使用的皮革类型。课程明确说明，这次更新不代表原材料或品质改变。

示例：

| 产品 | 旧描述 | 新描述 |
|------|--------|--------|
| Brooklyn Shoulder Bag | NATURAL FULL GRAIN LEATHER | GENUINE FULL GRAIN LEATHER |
| Quilted Tabby | GENUINE NAPPA LEATHER | GENUINE FULL GRAIN LEATHER |

建议回答口径：

这次皮革类型描述更新是为了统一不同产品中的用词，让顾客更清晰地了解 Coach 所使用的皮革类型。原材料在任何生产批次中始终保持一致，品质没有改变。这体现了 Coach 对高品质皮革的坚持与专业承诺。

## 常见顾客问题处理

| 顾客问题 | 回应方向 |
|----------|----------|
| 为什么皮革包价格高？ | 从进口天然皮革、认证制革厂、自有复鞣配方、严格检验四个角度解释。 |
| 为什么皮面有纹路？ | 说明天然牛皮会有疤痕和生长纹，Coach 保留天然纹理并通过检验。 |
| 为什么同款纹理不同？ | 说明每张皮革天然不同，后整理和裁切也会带来独特纹理。 |
| 为什么羊皮有毛孔？ | 说明天然羊皮经鞣制后会露出毛孔，Coach 会选择头层并做后整理。 |
| 为什么光滑皮革容易划？ | 说明柔软、较薄或丝滑效果的皮革更容易留下使用痕迹，可提供门店保养服务。 |
| 皮革描述词变了是不是品质降低？ | 明确不是，描述词更新是为了统一口径，原材料和品质没有改变。 |
