## 概述

本文档记录 `与Z同频，潮向未来` 课程中的 GC Signature Group 人才发展项目信息。该课程主要用于内部人才发展和管理者招募，不是顾客销售话术来源；在陪练系统中仅作为内部组织背景或管理类场景资料使用。

## 核心内容

### 1. 项目定位

GC Signature Group 是 GC 高潜力人才领导力发展项目。课程称第四届已启动报名，主题为“潮向未来”，目标是支持公司纳新 Gen Z 的策略目标，并支持内部人才持续发展。

课程表达的项目方向是：与 Z 同频，不只听你说，更带你玩；不只讲理论，更搞实战，从而建立与 Gen Z 之间真正的情感联结。

### 2. 项目培养目标

| 培养方向 | 课程说明 |
|----------|----------|
| 提升影响力 | 帮助店铺管理者打造个人品牌。 |
| 发掘优势 | 洞察成长机会。 |
| 挑战商业项目 | 通过高难度商业项目加速能力提升。 |
| 支持公司战略 | 携手公司实现变革中的更高目标。 |

### 3. 项目体验

参与 Signature Group 可以获得：

- 领导力和其他能力课程学习。
- 跨市场学习体验。
- 专属教练指导。
- 深入了解公司不同部门。
- 将经验沉淀为自己的管理语言与领导风格。

课程提到，从 2018 年至今，前三届 GC Signature Group 累计已有 68 位伙伴加入，涵盖 City Manager、General Store Manager 和 Store Manager 等不同职级。

### 4. 第四届报名条件

| 条件 | 课程说明 |
|------|----------|
| 职位 | 门店第一负责人，ASM 职位及以上。 |
| 任职时长 | 在 Coach 工作超过一年。 |
| 英语能力 | 对英语表达有信心，或有意愿学习。 |
| 表现要求 | FY25-FY26 期间带领团队表现优异，并有强烈自驱力。 |
| 后续流程 | 完成报名后，会收到后续竞争筛选流程。 |
