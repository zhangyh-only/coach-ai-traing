## 概述

Tabby 是 Coach 现代经典手袋系列。FY26 课程强调，随着 Tabby 趋势持续增长，该系列已经成为最能代表 Coach 品牌的标志性包款之一。Tabby 通过不同皮革材质、设计细节、尺寸、图案主题和背法，帮助不同顾客找到适合自己的选择。

## 核心内容

### 1. 系列定位

| 主题 | 课程事实 |
|------|----------|
| 品牌地位 | Tabby 被课程称为 Coach 明星产品家族之一。 |
| 设计定位 | Stuart Vevers 表示 Tabby 是现代 Coach 经典手袋。 |
| 选择逻辑 | 不同皮革、精致设计和丰富尺寸，让顾客找到适合自己的选择。 |
| 销售提示 | 课程强调“选 Tabby 准没错”。 |

### 2. 主题与设计故事

| 主题 | 课程重点 | 适合顾客 |
|------|----------|----------|
| Peanuts / 报纸印花 | 使用经典 Peanuts 漫画动作和报纸印花主题。 | 喜欢联名、漫画、趣味图案和收藏感的顾客。 |
| 涂鸦 / Sketch Print | 通过刺绣、微型吊饰、皮革压印等工艺让涂鸦印花更生动。 | 想要独特、手作感和艺术风格的顾客。 |
| Times Square / 纽约市 | 灵感来自纽约，从黄色出租车、比方说 I Heart New York 马克杯、自由女神像、大苹果等元素延伸。 | 喜欢纽约文化、城市纪念感和活泼主题的顾客。 |
| Western Tabby | 受西部风格启发，含铆钉腰带、流苏披肩外套、毛毡牛仔帽等灵感。 | 喜欢马术、西部、户外和复古感的顾客。 |

### 3. Chain Tabby

| 特点 | 课程信息 |
|------|----------|
| 上新节奏 | Chain Tabby 在 2025 年春季时装秀首次亮相，并作为春季重点。 |
| 材质 | 使用 Soft Grain Leather 柔软颗粒纹皮革。 |
| 手感 | 课程描述为垂坠柔软、手感如丝般柔软，并有自然纹理。 |
| 细节 | 标志性立体雕塑 C 金属配件、翻盖磁扣闭合、以皮革条包覆的内部设计、三条可拆卸肩带，其中包括链条。 |
| 背法 | 可通过不同肩带实现更多背法。 |
| 尺寸 | 课程展示小号、中号和大号，分别可容纳小型 iPad、中型 iPad、13 寸笔记本电脑。 |

### 4. Quilted Denim Tabby / Patchwork Tabby

| 款式 | 课程重点 |
|------|----------|
| Quilted Denim Tabby | 面向喜欢休闲、年轻氛围的顾客；材质让整体风格更随性。 |
| Quilted Denim Patchwork Tabby | 使用经典手袋结构，布料拼接工艺图案由旧丹宁面料拼贴而成，体现可持续理念和减少环境破坏的承诺。 |

### 5. Tabby Shoulder Bag 28

Tabby Shoulder Bag 28 采用更加小巧的廓形，并以精致的皮革、金属配件和镶嵌水晶提升华丽感。课程提示，该款适合搭配晚宴或约会造型。

内部结构卖点：款式较长，能容纳顾客晚上外出所需的一切物品。

### 6. Twisted Tabby

Twisted Tabby 的重点是弯曲、扭曲外观的 C 金属配件。课程强调，这款新款不对称包型本身受到关注，连金属配件也是扭曲设计。

### 7. Soft Tabby Bag

Soft Tabby 是 Coach 最畅销的软 Tabby 款式之一。FY26 课程强调该款重新设计出正面翻盖，以搭配春季时装伸展台上的 Chain Tabby Bag。新的翻盖有更简洁的皮革包边和 Bombé 皮革弧面饰边，取代旧款顶部缝线和边缘上色等细节。

### 8. Monochrome Tabby

Monochrome Tabby 进入渐层色时代，所有细节均为粉笔白色，包括缝线、金属配件和皮牌。课程提醒，染色搭配金属配件容易刮伤，因此导购在销售时要提醒顾客使用这些袋子时格外小心。

### 9. Knit Soft Tabby

Knit Soft Tabby 与真皮 Tabby Shoulder Bag 具备相同功能设计，但使用 72% 羊毛和聚酯纤维混纺提花织物，并使用 Bouclé 纱线编织，触感柔软蓬松。课程明确护理指南：只能干洗。

设计补充：超大 Signature C 针织图案的灵感来自时尚伸展台上的这款针织衫；Stuart Vevers 提到强化针织质感，让纹理更活泼。

## 销售引导原则

| 顾客偏好 | 推荐方向 |
|----------|----------|
| 想要经典、百搭、可多背法 | Chain Tabby 或 Soft Tabby。 |
| 喜欢轻松、年轻、丹宁风格 | Quilted Denim Tabby。 |
| 注重可持续或手作拼接感 | Quilted Denim Patchwork Tabby。 |
| 喜欢纽约、城市主题、趣味配件 | Times Square / 纽约主题 Tabby。 |
| 喜欢西部、马术、复古户外感 | Western Tabby。 |
| 想要晚宴或约会包 | Tabby Shoulder Bag 28。 |
| 喜欢极简同色系 | Monochrome Tabby，但需提醒护理小心。 |
| 喜欢柔软蓬松针织质感 | Knit Soft Tabby，并提醒只能干洗。 |
