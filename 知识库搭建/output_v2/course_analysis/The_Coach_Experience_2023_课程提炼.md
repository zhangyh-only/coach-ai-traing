## 一、CONNECT · 引领/接近随意浏览的顾客（p06-09）

**真诚自然的语言与礼仪（p06-07，接近前基调）**
- 语言沟通：没有脚本或公式，按实际情况个性化（例"午安/你好/新年快乐，欢迎来到 Coach～"）；避免勉强或不真诚的恭维。
- **55387 定律（p07）**：体态语言 **55%** / 语气 **38%** / 措辞 **7%** —— 非语言比字面更重要。
- **三要素（p07）= 微笑 + 眼神交流 + 姿势**。戴口罩时用眼睛微笑；眼神交流配微笑+点头；姿势影响顾客对店铺/品牌第一印象。

**引领顾客的四大方向（p08，四象限话术）**
- **新品**："这是本月推出的帆布手袋 Spin Tote，具防水及抗污功能，你可以看看。"
- **店铺区域**："店铺有三个区域：女装手袋区、男装手袋区和 lifestyle 区，请随意参观。"
- **每日/周重点产品**："这边是我们跟 Disney 合作的最新系列，现正期间发售，请随便试。"
- **顾客特征（FABS）**："留意到小姐背着 Parker 手袋，我们刚到货 Madison 系列最新款，有相似之处，请随便看看。"

**增加继续互动可能性（p09，按顾客状态三种话术）**
- 有回应："我叫 XXX，您先随意浏览，我过几分钟再来帮您。"
- 在看服饰/鞋履："如有需要我可为你提供合适尺码，您先随意浏览吧。"
- 在看手袋："我们不少手袋内都放有可拆卸长肩带，需要可替你调整长度，您先随意浏览吧。"

## 二、CONNECT · 观察 FABS 破冰（p09-10）

**FABS = 观察顾客身上的四类线索**（p10，注意：不是产品介绍法）：

| 字母 | 含义（p10） |
|---|---|
| **F** | Face 脸部 |
| **A** | Accessory 配饰 + 衣服 |
| **B** | Bag 手袋/购物袋 |
| **S** | Spoken 说的话 / Shoes 鞋履 |

- 应用提示（p10）：**"请具体地描述细节！"**、**"不用立即联系到产品！"**
- S 示范（p10）：偷听到顾客与朋友讨论热门打卡地标 →"您已经去了吗？我也想找时间去看看！"
- 观察目的（p09）："顾客的风格和反应可以为你提供有意义的见解，帮助你更有效地与其互动。"
- 观察后→提开放式问题（p10）："提出问题，尤其是开放式问题，是了解顾客及其需求的最关键工具。"

**开放式 vs 封闭式问题（p11-12）**
- **开放式（p11）**：邀请顾客敞开心扉、表达想法；用于探索需求；**5W1H** —— Who"今天帮谁选礼物？" / What"平常喜欢什么品牌？" / Where"想在什么场合用？" / When"婚礼什么季节？" / Why"为什么有这样的顾虑？" / How"习惯怎样搭配牛仔外套？"。提示：顾客不愿分享时，待其回应再问。
- **封闭式（p12）**：是/否、这个/那个等几字回答；三种用途 —— **缩小选择范围 / 澄清或确认需求 / 提示决定**（如"这两个颜色更喜欢哪个？""听起来您需要中性色，对吗？""这款适合您的需求，要不我帮您一起选？"）。

## 四、BUILD · 渲染产品 BFA（p15-16）→ 归 KB3

**A) 渲染产品的四个切入点（p15）**：皮革刺针烫印（Coach 皮革诞生七步骤）/ 五金及工艺（工艺七要点）/ Coach 经典元素与精品细节（七大经典元素）/ 设计灵感（Bonnie Cashin 经典设计、Stuart Vevers 设计灵感、季度产品灵感）。总述："通过分享 Coach 产品专业知识，让顾客对自己的选择更坚定、更有信心！"

**B) BFA 介绍法（p16，把产品讲成对顾客的价值）**：
- **B = Benefits 好处** —"完美满足日常外出放置物品的需求"
- **F = Features 特征** —"廓形方正，容量大，内部收纳空间十足"
- **A = Advantages 优势** —"功能性十足，轻松放置多种个人物品"
- 应用提示：先找出顾客真正的消费需求（站在顾客角度、找产品亮点、明确功能、内容整合）。
- 完整范例（妈妈给留学女儿买包）：B"这款 Cashin Carry 手袋任何场合都能打造完美造型" / F"轮廓方正，包型简约" / A"线条干练不易过时；经久耐用，可一代代传承"。

## 五、CLOSE · 价格三明治（p17-19）→ KB3 扩写

- 引子（p17）："多少钱？"是顾客最常问的，但只用价格回答时顾客会怎样反应？
- 价值观铺垫（p18）：**价格 Price** = 需支付的金额（数字，标在标签/网站）；**价值 Value** = 顾客对商品渴求程度的反映（愿付多少交换）。"注意对价值的理解如何影响对价格的理解。"

**价格三明治三层（p18，自上而下）**：

| 层 | 课程要点 | 示范话术（p18） |
|---|---|---|
| ① **介绍产品** | **使用正确的官方名称** | "这是最新的 Denim Soft Tabby 手袋。" |
| ② **分享产品价格** | （价格夹在中间） | "它的价格是 19800。" |
| ③ **渲染产品** | 材质/工艺/设计灵感/风格等 | "采用最新一代丹宁材质，由有机棉制成，践行可持续概念，触感柔软且耐磨。" |

- 含折扣进阶范例（p19，Town Bucket）：①"本月最新的 Town Bucket 手袋" → ②"定价 19800，正做对折活动，折后 9900" → ③"磁吸式闭合设计，轻松拿取，轻巧实用"。
- **与现有库对照**：现有"名称→价格→工艺特色"三层顺序一致；课程版更全 —— 第①层强调**官方正名**，第③层可含材质/工艺/设计灵感/风格/可持续多维度，并给了**带折扣价**的中间层处理。**据此扩写现有篇。**

## 六、CONTINUE · 售后维系（p19-32）→ 仅取对客动作进 KB3

**入 KB3 售后维系（对客动作，与陪练相关）**：
- **Clienteling 1-2-3（p20-25）**：1. 在顾客档案添加标签/描述 → 2. 致以个性化感谢/播种 → 3. 跟进播种、邀请回店。时间线：第1天→第1周→第2周→第3个月全渠道联系。数据点：与销售顾问建立个人关系的顾客，钱包份额近乎其他品牌的 **2 倍**。工具 WeClient Pro 企业微信；播种内容 ≤200 字、仅创建者可见。

**不入陪练 KB（内部运营/CRM/学习平台，与 AI 顾客对练无关，仅作背景）**：
- Client Loop（p26-29）：全球顾客回馈计划、问卷时间线、NPS 净推荐分（Detractor 0-6 / Passive 7-8 / Promoter 9-10）、OSAT、72 小时回复警讯。
- WeLearning（p29-32）：员工学习平台（产品基础/时尚概览/TCX/新手必修/互动社区/我的培训/店铺辅导/公司信息）。
