## 【客户心理与沟通原则】

### 消费心理 / 驱动机制
- **GenZ / Hobo 潮流驱动**：课程明确点出"不晓得是不是 Gen Z 潮流的关系，大家进来都纷纷选购简约利落的 Hobo 手袋，让自己的个人风格焕然一新"——顾客买的是"用一只简约包焕新个人风格"的自我表达。
- **安全色 vs 新鲜感的张力**：很多顾客习惯/偏好黑色手袋，光泽皮革的"自然色"被定位为黑色的"很棒的替代品"——满足"想要不一样但又怕踩雷"的心理。
- **审美偏好**：垂坠感、柔软触感、复古做旧感、闪亮釉面光泽是核心吸引点（"顾客一定会深深着迷"）。

### 沟通风格
- 用感性、向往式语言传递材质魅力（"令人惊艳""无可挑剔""深深着迷"），契合 GenZ 与都市悦己型的情绪价值需求。
- 从"包"延伸到"整体光泽造型"，引导顾客想象穿搭场景（街头穿搭灵感）。

### 沟通禁忌 / 红线
- 不要把"光泽皮革(Glazed Leather，牛皮+釉)"直接等同于"漆皮(Patent Leather)"——课程仅在搭配层面关联两者，未声明同材质，勿编造材质等同关系。
- "做旧复古感"是设计意图而非质量缺陷——讲清楚是工艺特意呈现，避免顾客误读为旧/磨损。

### 品牌价值传递方式
- 以"意大利原产地 + 独家工艺(滚筒摔软+上釉) + 核心棒球手套鞣制皮革垂坠感 + 立体雕塑 C 标识"承载轻奢与专业，呼应 Expressive Luxury 真我新奢的自我表达内核。
