## 概述
A.L.L. 是 Coach TA Card（训练卡）方法论族中的一套全渠道销售技巧，隶属 The Coach Experience 销售旅程的 **BUILD（打造）** 阶段。它解决的核心问题是：当顾客以在线、实体、社群媒体等多种方式购物时，导购如何以"全渠道销售顾问"姿态为其创造完美流畅（无缝）的购物体验，并把单次接待延展为长期顾客关系。本文档供陪练 AI 顾客判断学员是否真正"跳出本店货架"协助自己、供对话评估引用 A.L.L. 三步是否执行到位。

## 方法论定位与适用场景
- **所属方法论**：TA Card（Coach 训练卡）；**A.L.L. 技巧 = ASK / LEVERAGE / LOOK 三法则**。
- **所处销售旅程阶段**：The Coach Experience 旅程环 **CONNECT → BUILD → CLOSE → CONTINUE** 中的 **BUILD（打造）** 阶段。
- **映射线下 6 阶段**：对应**需求诊断 / 触感体验 / 价值传递**环节——用 A.L.L. 打造体验、扩展可推荐范围。
- **适用行为类型**：多类型通用（适用于所有进店顾客）；其中"未来消费者"区分暗含培育型 / 长期关系型顾客。
- **适用渠道**：**全渠道**（线上 + 实体打通，含社群媒体）。本卡右上角标注 **CR**（收银机图标），表明为 **Retail 正价店（CR）** 场景版本；技巧本身要求把视野"拓展至实体店铺之外"，覆盖在线 / 实体全渠道购物体验。
- **使用时机**：顾客需求未明确、询问其他渠道见过的产品、或心仪款店内无货时，用 A.L.L. 把对话从"本店货架"扩展到"全渠道资源 + 长期关系"。

## 核心要点 / 步骤拆解（A.L.L. 三法则）

### A — ASK 询问问题
- **动作**：通过提问理解顾客需求；先问后推，而非直接推销 / 报款。
- **要点**：进店初期顾客需求常不清晰，需被引导表达。这一步是后续"运用资源"和"跳出框架"的前提。

### L — LEVERAGE 运用现有资源
- **动作**：调用现有资源协助顾客找到他们想要的产品。
- **可调用资源（原文保留）**：**WeClient**、**小红书**、**Link to EC**。
- **要点**：社群媒体与社交电商是全渠道触达的关键载体；不应只凭店内货架在售款回应顾客。

### L — LOOK 跳脱实体店铺框架
- **动作**：了解不同店铺或渠道的产品及服务；掌握最新信息；**建立你的 Coach 社群**。
- **要点**：把视野从本店扩展到跨店 / 跨渠道，能回应顾客对"其他渠道产品"的询问；"建立 Coach 社群"承载长期关系价值，呼应 CONTINUE 阶段。
- **易混淆提示**：第二、三个 L 含义不同——**L**EVERAGE（运用资源）≠ **L**OOK（跳出店铺框架），勿混为一谈。

### 贯穿原则：区分"现在购买"与"未来消费者"
- 顾客分两类心智状态：**"现在购买"**（即时成交需求）与**"未来消费者"**（尚未购买、需被培育的长期关系对象）。
- 课程要求区别对待：对"未来消费者"不催单，而是**埋下种子**建立长远关系。
- 缺货 / 产品无法取得时，**不应让"店内无货"成为对话终点**；应转向跨渠道方案与长期关系培育。

## 适用边界 / 红线（原文标注）
- **只局限店内商品**（卡片以"？"标注）：技巧仍需聚焦店内可售 / 合规商品；跨渠道协助的落点仍须回到店内可售商品。跨渠道调货、外部渠道下单、推荐外部链接的具体执行口径课程未给出，**勿编造承诺**。
- "全渠道"不等于"无边界"——技巧仍受"只局限店内商品"约束。

## 话术参考
> 本 TA Card 为方法论与演练卡，**未提供具体成交话术 / RPT 录音原话**。下列为卡片中可保留原味的"方法论说明语 / 演练讨论提问"原文（非顾客应对话术）。落地成交话术需另行从 RPT 类资料补全。

**A.L.L. 理念表述（原文）**
- 「不管顾客选择怎么购物，无论是在线、实体，还是透过社群媒体，作为全渠道销售顾问，我们都要为他们创造完美流畅的购物体验。有了 A.L.L. 法则，这会变得轻松简单。」
- 「记住询问问题，运用现有资源，并将视野拓展至实体店铺之外！运用你的资源当中的 A.L.L.，就能协助顾客得到他们想要的东西（以及更多！）。」

**带教引导提问（面向销售顾问，开放讨论，无标准答案）**
- 「顾客进店询问另一个渠道的产品时，如何推荐你店铺现有的类似产品？」
- 「运用哪些工具来替顾客找出他们正在寻找的产品？」
- 「如果他们想要的产品无法取得，如何为未来跟进埋下种子？」
- 「如何看出'现在购买'与'未来消费者'之间的区别？针对后者我们如何更进一步建立长远的关系？」

## 角色演练机制（TA Card 通用演练框架）
- **三角色**：销售顾问 > 顾客 > 观察者。
- **演练前界定**：谁是销售顾问 / 顾客 / 观察者；角色扮演的开始与结束点；要关注哪些产品。
- **演练后反馈讨论**（按 销售顾问 > 顾客 > 观察者 顺序复盘）：「在这技巧中什么做得好的？」「在这场景 / 技巧中，什么可做得更好或不同？」

## 常见误用
- **跳过 ASK**：不问需求直接推销 / 报款，使后续资源运用和跨渠道扩展失去依据。
- **只看本店货架（LEVERAGE 缺失）**：仅凭店内在售款回应，未利用 WeClient / 小红书 / Link to EC 等任何线上 / 社群资源。
- **视野困在本店（LOOK 缺失）**：无法回应顾客对其他店铺 / 渠道产品的询问，不掌握最新信息。
- **缺货即终止对话**：心仪款店内无货时直接说"没有"打发顾客，未推荐类似款、未为未来跟进埋种子，造成流失。
- **对所有顾客一刀切催单**：未区分"现在购买"与"未来消费者"，对待培育顾客硬推成交，错失长期关系。
- **越界承诺**：以"全渠道"为名承诺非店内 / 不合规的商品或渠道，踩"只局限店内商品"红线。

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客据此对学员是否真正"跳出本店"作出不同反应：
- 学员先 ASK 理解需求、再用 LEVERAGE 调用资源、用 LOOK 回应跨渠道询问 → 顾客感到被认真对待、愿意推进。
- 顾客询问"在别处见过的款"或心仪款店内无货时，学员若直接说"没有 / 我们这没这款"就打发 → 顾客失望、降低体验感甚至流失；学员若推荐店内类似款并提出后续跟进 → 顾客（尤其"未来消费者"心智）愿意留联系方式 / 接受长期关系。
- 触发"未来消费者"扮演：顾客本次未必成交，期待被培育而非被催单。

### 对话评估引用
作为 A.L.L. 技巧运用是否到位的判断依据：
- ASK / LEVERAGE / LOOK 三步是否执行（先提问→调资源→跳出本店框架）。
- 是否守"只局限店内商品"边界、未越界承诺。
- 缺货时是否推荐类似款 + 为未来跟进埋种子，而非让对话终止。
- 是否区分"现在购买"与"未来消费者"并对后者经营长期关系 / Coach 社群。
- （带教演练场景）三角色与起止点、关注产品是否界定清楚。
