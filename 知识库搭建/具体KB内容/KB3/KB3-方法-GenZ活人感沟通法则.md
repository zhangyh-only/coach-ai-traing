## 概述
本文档整合 CoachX 无届体验两门 Gen-Z 课程的核心沟通方法论，沉淀一套"读懂 Gen-Z、用活人感与真诚连接这代人"的沟通法则：活人感聆听-回应两步法、夸夸 SBI 正向反馈、SBIR/SBIRB 改进型反馈，以及背后的 Gen-Z 冰山模型（体验感/意义感/安全感）。它服务于陪练系统对"沟通是否有活人感、是否带观点、是否用提问代替批评"的扮演与评估判断。

> **重要边界（须先读）**：两门源课程均为**管理者带教 Gen-Z 员工**向的内部沟通工作坊（受众为 SIC/MM/店长等带教角色），**不是面向顾客的销售流程课**。其方法论与 Gen-Z 心理模型可**迁移**到"对客沟通"（尤其面向年轻段的都市悦己型/质感自用型顾客），但迁移属推断，须经业务确认——详见"来源与待确认事项"。本文档中凡话术示例多为"管理者对员工"语境，仅作沟通结构范本，**不可直接作为对客话术复读**。

## 方法论定位与适用场景
- **所属体系**：CoachX 无届体验（Beyond Experience）联合培训项目。本课方法论为 SBI/SBIR 通用反馈模型 + 冰山模型在 Gen-Z 场景的应用，**非** Coach TA Card 体系（不是 E.N.D./A.L.L./A.A.A./价格三明治），下游归类勿张冠李戴。
- **对应行为类型**：Gen-Z 群体通用心理，最贴近**都市悦己型**与**质感自用型**中的年轻顾客（映射属推断，待人工确认）。
- **渠道**：课程不区分 Retail/Outlet，全渠道通用。
- **迁移到销售流程的环节（推断）**：
  - 活人感聆听-回应 → 迎宾破冰 / 需求诊断（倾听情绪与价值观、建立真诚连接）
  - 冰山模型"理解动机" → 需求诊断前的共情铺垫
  - 夸夸 SBI / SBIR 反馈 → 一手用于团队带教与售后复盘，对客可迁移为"带观点的正向肯定"

## 核心要点 / 步骤拆解

### 一、Gen-Z 冰山模型（沟通前提：先看懂动机）
两门课一致传递的底层模型——可见行为只是表象，要先看懂"冰山下的核心需求"才能有效沟通。

- **冰山下（成长环境，塑造这代人的根因）**：教育资源大爆炸/井喷、数字时代狂飙（数字原住民、社媒为日常）、全球化大浪潮/大熔炉。
- **冰山上（外显行为表现）**：5G/SG 冲浪体验官（追新鲜、信息获取快）、真实悦己·微自由、情绪自由·微傲娇、职场行为有自己的态度、真实勇敢做自己。
- **冰山核心（三大底层需求，反复强调）**：
  - **体验感**：重视过程中的体验与感受，非仅功能。
  - **意义感**：在意所做之事 / 所买之物的意义与价值表达。
  - **安全感**：需要被尊重、被理解、心理安全。
- **核心心理金句（3547）**：「**只有当 Gen-Z 被听见时，他们才会听你说的。**」（Until they feel heard, they will not be able to listen to you.）——被倾听是建立信任与影响力的前提。

### 二、活人感互动（聆听 → 回应 两步法）
课程自命名的"活人感沟通"，核心是非模板化、真诚、不套话、带真实情绪、可适度幽默；前提是"以好奇心代替偏见"。

1. **Step 1 聆听**：捕捉冰山下的「感受、观点、期待」三层。
2. **Step 2 回应**：验证听到的「感受 / 观点 / 期待」是否准确，而非假设。
3. **回应句式模板（课程原文，两门一致）**：
   - 感受：「我能感受到你……」
   - 观点：「我能感受到你（认为/觉得）……」
   - 期待：「我能感受到你（期待/希望/想要）……」
4. **辅助原则**：**允许留白**（回应后给对方表达和思考的空间，不急于填满或下结论）、适时提问。

### 三、夸夸 SBI（正向反馈）
给 Gen-Z 正向反馈/表扬时用。SBI = Situation 情景 / Behavior 行为 / Impact 影响（通用反馈模型；3547 课程呈现序为 情景→影响→行为，命名"夸夸 SBI 技巧"）。

- **Situation 情景**：双方都知晓情景时**可简短或省略**（如"你刚刚""你休个假回来""你昨天发的小红书……"）。
- **Impact 影响 — 展现观点的夸夸**：**带有观点的肯定，不仅停留在事实**——这是本技巧的核心金句。
  - 事实（仅陈述，不够）：如"最近你 NPS 分数挺高""这月业绩全店第一"。
  - 观点（要叠加）：如"状态不错！""真棒！""藏不住的优秀！！""太细节了！"。
- **Behavior 行为 — 关注行为的提问（3547）**：看见背后的努力、强化具体正向行为（如"你是怎么做到的/怎么想到的？"）。
- **核心要点**：从"陈述事实"升级为"事实 + 观点 + 情绪"，让被夸者感到被真正看见（满足"被看见"需求）。

### 四、SBIR / SBIRB（改进型反馈）
给 Gen-Z 改进/纠偏反馈时用，核心是**不直接指责/下定论，用提问代替批评**，保护"安全感"需求。

- **3546 命名 SBIR**（Situation/Behavior/Impact/Request），强调 **"B 之 B 问 B 答"**——在 Behavior 行为环节用**两次提问**代替批评：
  1. **Situation 情景**：客观描述场景，不带评判。
  2. **Behavior 提问①（澄清并回应）**：用提问澄清行为背后原因（"发生啥了，是不是遇到什么事情了呀？"），并共情回应（"我能理解线上有很多顾客消息需要回复"）。
  3. **Behavior 提问②（共创行动）**：与对方一起商量改进方案（"那我们一起想想，以后再遇到类似情况怎么处理比较好？"）。
  4. **Impact 影响**：客观说明该行为对结果的影响（如对现场顾客感受、NPS 的影响）。
  5. **夸夸（注入信心）**：以鼓励收尾、表达信任（"我完全相信你再遇到一定可以做好！"）。
- **3547 改进型核心**：**以结果为导向、共情 + 提问启发**——先共情、问原因、再共商更好做法，而非指责说教。课后作业出现扩展版 **SBIRB（问B答）**，课程正文未给全称定义（待补，详见待确认事项）。

### 五、有效反馈三要素（3547）
任意反馈场景通用：**具体的 / 即时的 / 平衡的**（学员补充理解：基于事实、有同理心、正负平衡、当下且看得见）。

## 话术参考
> 以下原话来自课件示例气泡，保留口语原味。**多为"管理者对员工"语境，作沟通结构范本，不可直接对客复读。**

### 活人感互动 · 回应句式（对 Gen-Z 通用）
- 感受：「我能感受到你……」
- 观点：「我能感受到你（认为/觉得）……」
- 期待：「我能感受到你（期待/希望/想要）……」

### 活人感互动 · 示例对话（3546，应对"每天早起化全妆"的 Gen-Z）
- 顾客原话：「对呀，因为每天都想美美的……假睫毛是灵魂，必须要支棱起来，化妆就是我生活的一部分！」
- 回应-感受：「我能感受到化妆能点亮你，带给你好心情……」
- 回应-观点：「我能感受到假睫毛对你来说，是化妆的点睛之笔啊，为什么是睫毛而不是口红呢？」
- 回应-期待：「我能感受到你对美是有追求的，那你休息天出门也会化妆吗？」

### 夸夸 SBI · 正例（事实 + 观点）
- 改造前（仅事实）：「这个月 tabby 卖的很好啊」
- 改造后（事实 + 观点）：「这个月 tabby 卖的很好啊，继续努力呀，"tabby 大将军"！！」
- 带观点的肯定示例：「状态不错！」「真棒！」「藏不住的优秀！！」「太细节了！」
- SBI 完整示范（3547）：「你昨天发那个小红书火了欸！我看有 100+ 点赞了……你那图拍得真的超绝！」+ 行为提问「哎你觉得这条最会火？」

### 改进型反馈 · 正例（结果导向 + 共情提问）
- 「来，我们以结果为导向来思考一下，我们可以做些什么？」
- 「……可以说说取消单这么多有什么原因吗？」（先问原因再共商，不直接定性）
- SBIR 行为双问（3546）：「一直低头在看手机，发生啥了，是不是遇到什么事情了呀？」→「那我们一起想想，以后再遇到类似情况怎么处理比较好？」

### 反例 · 应避免（Left 模板化/指责，3547 标注为反例）
- ✗ 空洞夸赞：「真棒！！」「优秀！！」「辛苦了！」（停留事实、无具体无观点）
- ✗ 指责说教：「你别自己闷着，浪费时间」「我希望这种情况不要再发生了好吗？」
- ✗ 带偏见预设：「最近晚上是不是又熬夜打游戏」（违背"以好奇心代替偏见"）

## 对客沟通禁忌（迁移到对客场景的红线，源自新场景资料）
> 下列禁忌来自业务老师 docx《coach陪练知识库建设相关文件》，针对**质感独行侠/质感自用型**等顾客，与本课"活人感、真诚不套路"的沟通基调一致，作为对客迁移时的红线落点（同步进 KB1 沟通规范 / KB4 被推开点 / KB5 扣分项）：
- **群发营销短信**：违背"有温度不打扰"，离店后应一对一感谢 + 保养指南，**不群发**。
- **过度热情、贴身跟随**：压迫感会推开追求自我空间的顾客，违背"允许留白"。
- **主动提折扣满减**（除非顾客主动问）：把价值沟通降维成价格博弈，削弱"质感投资/意义感"。
- **用"网红爆款"作推荐理由**：质感自用型/独行侠反感被当成只追潮流的人，须给"经典/工艺/意义"的理由而非爆款噱头。

## 常见误用
- 把 SBI（用于夸赞）与 SBIR/SBIRB（含 Request、用于改进反馈）混用；二者都以"夸夸/注入信心"收尾，但用途不同，勿张冠李戴。
- 正向反馈只罗列事实数据、无观点无情绪（被夸者感受不到被真正认可）。
- 改进反馈直接指责/下定论（"你不该……"），跳过"提问澄清 + 共创方案"，踩"安全感"红线。
- 活人感互动只顾输出、抢话、急于成交，不给"留白"，不验证理解就下结论。
- 把 Left 反例话术误当正面话术沉淀（3547 中 Left=反例、Right=正例）。
- 用套路化、无温度的标准话术对待 Gen-Z（违背"活人感"），或带偏见预设代替好奇提问。
- 把课程举例语境词（tabby/soft quilted tabby/UPT/add on/NPS）反推成产品或考核参数。

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客对学员沟通"有没有活人感"作出不同反应：学员先聆听、用"我能感受到你……"复述验证其感受/观点/期待并允许留白 → 顾客感到被听见、愿意敞开；学员套路化、不接情绪、急于推销或带偏见预设 → 顾客（年轻段都市悦己型/质感自用型）变冷淡、疏离。正向肯定带观点+情绪时打动，空洞"真棒/优秀"无感。踩对客禁忌（贴身跟随/主动提折扣/网红爆款理由）→ 触发防备或被推开。

### 对话评估引用
作为"Gen-Z 活人感沟通是否到位"的判断依据：是否先聆听捕捉感受/观点/期待并复述验证、是否允许留白、正向反馈是否带观点而非仅陈述事实、改进/共情是否用提问代替指责、是否以好奇心代替偏见、是否避免群发/贴身跟随/主动提折扣/网红爆款理由等对客禁忌。
