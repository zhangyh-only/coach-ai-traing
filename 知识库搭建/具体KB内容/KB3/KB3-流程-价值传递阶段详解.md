## 概述
**所属四阶段：BUILD（打造）** —— The Coach Experience 四阶段 CONNECT → BUILD → CLOSE → CONTINUE 中，BUILD 对应本项目"触感体验 + 价值传递"，本篇为其后半（价值传递·故事加持）。（注：价格三明治在课程中归 CLOSE 卡，业务老师骨架将其放在"价值传递→主动成交"衔接处，本篇按骨架在报价中使用，见"来源与待确认事项"。）

价值传递是 Coach 线下正价店 6 阶段销售流程的第 4 阶段，位于"触感体验"之后、"主动成交"之前。核心任务是在顾客已经摸过、试过、对单品有了感官认知之后，给她一个"值得买、且适合她"的理由——用品牌经典传承、工艺与服务权益，把"商品"抬升为"对的那一只"，让顾客在心里完成"价值＞价格"的认同。它是 POC 门店反馈中"话术不够漂亮、打动不了顾客"问题最集中的环节，直接决定陪练里 AI 顾客是否被说服、以及评估中"专业性：知识运用与价值传递"维度的高低。

## 阶段 / 场景定义
- **所属流程**：线下正价店 6 阶段（迎宾破冰 → 需求诊断 → 触感体验 → **价值传递** → 主动成交 → 售后维系）。私域 / 奥莱快速成交链路把价值传递压缩在"组合优惠 + 经典不过时"几句内带过，属渠道变体。
- **适配渠道**：线下正价店 Retail（主）；微信私域 / 品牌奥莱为压缩变体。
- **所处阶段**：价值传递（第 4 阶段，故事加持）。
- **核心目标**：让顾客认同"这款很适合您"而非"很好卖"——用经典传承（设计灵感不过时）+ 服务权益自然植入（免费刻字 / 护理）+ 工艺与品牌叙事，建立"价值＞价格"的认知，为报价与成交铺垫；处理报价过程中出现的价格 / 价值存疑异议。
- **销售强度**：中（动作核心是"分享与认同"，不是施压成交；按业务老师骨架"这款很适合您"而非"很好卖"，强度应低于主动成交阶段）。

## 客户状态描述
- **进入方式**：顾客刚经历触感体验，已上手摸过皮质、试背照过镜、看过五金 / 内里 / 背带细节，对单品有了具体感官印象，但还没下决定。
- **需求清晰度**：基本款式方向已锁定 1-2 款（需求诊断阶段已完成匹配），此刻的核心问题从"要哪款"转向"值不值、是不是我的"。
- **情绪与戒备点**：
  - 价格存疑：心里在算"这个包是不是太贵了""贵在哪、值不值这个价"（2189 价格三明治技巧的设计前提）。
  - 价值认知不足：容易"只看价格、把决策聚焦在数字上"，需要被引导理解价值＞价格（2189）。
  - 不想被当成只看 logo / 只图炫耀的人：年轻一代购买动机从"炫耀品牌"转向"自我表达"，反感把奢华窄化为"贵 / 大牌"（2047）。
  - 怕过时 / 怕选错：在意"能背几年不过时"，对当季趋势单品还会担心"追这个潮流会不会很快过时"（3671）。
- **配合度**：愿意听，但只对"专业且真诚、像朋友分享"的内容有正反馈；一旦听到空洞的"大牌就是贵 / 很好卖 / 网红爆款"或端着的官方客服腔，会冷淡、把注意力锁回价格数字。
- **渠道差异**：奥莱 / 私域顾客（性价比入门型为主）此阶段更关注"经典款耐用、不到正价一半"的组合价值，价值传递更短、更直接落到"经典不过时 + 划算"。

## 标准流程
> 业务老师骨架（A1·价值传递）三步主线：**经典传承（设计灵感不过时）→ 服务权益（免费刻字 / 护理自然植入）→"这款很适合您"而非"很好卖"**。下列在此骨架上用课程事实补全动作与方法论，体现"故事加持"的体验动线。

1. **经典传承 / 故事加持（设计灵感不过时）**：把单品放回品牌叙事——讲设计灵感、经典传承、工艺与季节企划，让顾客感到"这不是一时爆款，是经得起时间的设计"。可用的素材：品牌理念 Expressive Luxury（真我新奢，2047）；本季趋势的"本季流行 X + Coach 如何诠释 Y"双段式叙事与设计总监 / 媒体语录背书（3671）；档案灵感"回溯品牌档案 + 现代演绎"的传承＋创新叙事（3671）。**沟通逻辑：情绪先于理性**——先用品牌传统与工艺渲染、勾起情绪共鸣，价值认同再建立在情绪之上（2189 核心原理）。

2. **工艺与材质价值承托（报价用价格三明治）**：当需要带出价格时，用 **价格三明治技巧**（2189）：按"产品名称 → 价格 → 工艺及特色"三层组织，把价格夹在产品名称与工艺特色之间，避免裸报价的价格冲击；陈述时强调产品的价值、特色与优势，把对话重心放在价值而非价格上。工艺 / 材质事实须引用 KB2 真实产品数据（来源 LWG 金银级、独家复鞣、一次裁一个包、FFP 检验等口径见 KB2-正价-皮革与材质知识 / KB4-画像-质感自用型），不得编造价格或工艺数字。

3. **服务权益自然植入（免费刻字 / 护理）**：把增值服务作为"附加价值"在价值传递中自然带出，而非到收银台才补——
   - **Coach 压印服务**：吊牌可做个性化压印（刻字母 / 纪念日期），"会更像一只专属的包包"，强化专属 / 独一无二感（3708）。
   - **Coach 护理服务**：用一段时间后可到任意 Coach 门店免费清洁与基础护理，"相当于一直有人帮您照顾它"（3708），承接顾客"贵的包用旧了没人养护"的隐忧。
   - 红线：保修仅"全国门店一年保修，含拉链 / 五金配件等"，护理仅"免费清洁和基础护理"，刻字 / 退换的费用与条件课程未给出 → 不夸大、不编造（3708；具体口径见 KB1 服务 / KB3-售后维系）。

4. **个性化认同（"这款很适合您"非"很好卖"）**：把价值点收束到"为什么适合这位顾客"。按 2189 进阶要求——先了解顾客的生活方式与需求，再定制最能激发她渴望的那个价值点；不同画像渴望点不同（质感自用型听工艺 / 耐用 / 独一无二，都市悦己型听趋势 / 自我表达 / 专属，礼品选购型听"够正式 / 对方会喜欢"，性价比入门型听"经典耐用 / 划算"）。落到一句真诚的"适合您"，呼应品牌宗旨"激励顾客自信探索自己的各个面向、做真实自己的勇气"（2047），而不是把包说成"很好卖 / 大家都买"。

5. **应对价值 / 价格存疑（顺势处理报价中异议）**：报价后若顾客对价格或价值产生疑问，在本阶段用"不谈便宜谈值"承接（详见下一节"该阶段典型异议及应对"）；若升级为真实价格异议 / 拖延 / 比价，则进入 KB4 异议体系与主动成交阶段策略。

> **渠道变体（奥莱 / 私域，压缩链路）**：价值传递浓缩为"经典款耐用、不过时 + 组合优惠划算"两句，强调"正品保障 / 经典不过时 / 价格不到正价一半"，快速过渡到成交（业务老师 A2 性价比入门型应对方向 + 画像 B 营销口径）。

### 把产品讲成对顾客的价值：BFA 介绍法（The Coach Experience BUILD，p16）

价值传递的核心动作是"把产品讲成对这位顾客的价值"，The Coach Experience 给出 **BFA 介绍法**（注意：BFA 是渲染产品的介绍法，与 CONNECT 阶段观察顾客的 FABS 是两个不同缩写，不可混用）。三要素自下而上把"产品特征"翻译成"对顾客的好处"（来源：课程提炼 p16）：

| 字母 | 含义（p16） | 课程示例（p16，可引用，演练时换真实产品） |
|------|------------|---------------------------------------------|
| **B = Benefits 好处** | 对顾客的最终好处（先讲，最打动人） | "完美满足日常外出放置物品的需求" |
| **F = Features 特征** | 产品本身的客观特征 | "廓形方正，容量大，内部收纳空间十足" |
| **A = Advantages 优势** | 由特征带来的相对优势 | "功能性十足，轻松放置多种个人物品" |

- **应用前提（p16）**：先找出顾客真正的消费需求——站在顾客角度、找产品亮点、明确功能、内容整合，再用 BFA 组织语言。这与本篇标准流程第 4 步"个性化认同（这款很适合您）"同源：先听懂需求，再挑最打动她的那个价值点。
- **完整范例（p16，妈妈给留学女儿买包）**：B"这款 Cashin Carry 手袋任何场合都能打造完美造型" / F"轮廓方正，包型简约" / A"线条干练不易过时；经久耐用，可一代代传承"。
- **BFA 与价格三明治衔接（p16-18）**：BFA 解决"怎么把产品讲成价值"，价格三明治解决"怎么把价格夹在价值里报出来"——报价时把 BFA 讲出的好处/优势填进价格三明治的第①层（产品名称后的价值铺垫）与第③层（价格之后的工艺渲染），让价格被价值包裹。方法细节分别见 [[KB3-方法-BFA产品介绍法]] 与 [[KB3-方法-价格三明治技巧]]，本篇只讲"在价值传递阶段如何把两者串起来用"。

> 品牌故事 / 价值观素材：价值传递的"经典传承 / 故事加持"所需的品牌理念（Expressive Luxury 真我新奢）、品牌宗旨（COURAGE TO BE REAL 勇于求真）与四大价值观（坦于本心 / 匠心传承 / 真我同行 / 探索求新），统一引用 [[KB1-品牌-Expressive Luxury与品牌价值观]]，本篇只取其"作为价值传递的品牌锚点"用法，不在此重复定义品牌叙事。

## 该阶段典型异议及应对
> 承接 KB4 异议总表的"按环节分布"：**报价前=体验阶段、报价中=价值传递、报价后=成交推进**。本阶段聚焦"报价中"出现、围绕价格与价值认知的异议；只写关切主题、潜台词与应对方向，不写剧本台词。

| 关切主题 | 顾客潜台词 | 应对方向 | 关联行为类型 |
|----------|------------|----------|--------------|
| 价格与想的不一样 / 怎么定价 | 这么贵到底值不值？我只看到了价格 | 不谈便宜谈值：用产品 / 服务 / 使用价值三维撑价；价格三明治把价格夹在工艺特色之间，重心放在价值（业务老师"报价中"+ 2189） | 多类型通用（性价比入门型 / 都市悦己型尤甚） |
| 皮质 / 工艺溢价存疑 | 贵在哪？凭什么这个价，别只跟我说"大牌" | 给专业理由：来源（进口天然皮 / LWG 金银级）→工艺（独家复鞣 / 一次裁一个包）→检验（FFP），而非"大牌很好卖"（KB4-质感自用型 + KB2-皮革与材质知识） | 质感自用型 |
| 奢华是否等于炫耀 / 溢价 | 我买它是为了表达自己，不是攀比，别只跟我讲牌子贵 | 落到 Expressive Luxury 自我表达内核，把价值传递成"做真实的自己 / 适合您"，而非品牌身份与稀缺（2047，理念反推，非课程明文异议） | 都市悦己型 / 性价比入门型 |
| 描述词 / 标识变化引发品质存疑 | 皮革压印措辞怎么变了，是不是偷偷换了便宜材料、品质缩水了 | 先肯定关注 → 说明仅为统一不同产品用词 → 保证原材料任何批次始终一致、品质没有任何改变 → 收束到 Coach 对高品质皮革的坚持（3296 标准话术；红线：不得承认 / 暗示换料或降质） | 质感自用型 / 性价比入门型 |
| 解构 / 破损 / 手工痕迹像瑕疵 | 这种破破的、拆线的设计是不是没做好、会先坏 | 正向归因为本季解构设计语言 / 手工匠艺 / 独一无二，不含糊承认"可能没做好"（3671；红线：不得说成质量瑕疵） | 质感自用型 / 都市悦己型 |
| 趋势单品怕过时 | 追这个潮流会不会很快过时、休闲款撑不撑得起这个价 | 用"经典传承 + 设计灵感不过时"叙事，或"本季流行 X + Coach 如何诠释"把单品接回品牌长期主义，草编 / 果冻等休闲款用"高度耗时工艺 / 奢华质感"抬升，勿自降为便宜随便款（3671 + 业务老师"经典传承"骨架） | 都市悦己型 |

## 关键动作与话术参考
> 口语化、贴 GenZ、平等、带朋友感；品牌"不端着像朋友分享"。以下区分"课程可引话术原文（标来源）"与"调性 / 结构提示（非台词，演练时以真实产品填充）"。

**可引用的课程话术原文（标来源，引用时按真实产品 / 服务语境衔接）：**
- 服务权益植入（护理，来源 3708）：「这个包之后如果用了一段时间，您可以直接带到任意一家 Coach 门店，都可以免费帮您做清洁和基础护理，相当于一直有人帮您照顾它」
- 服务权益植入（压印 / 专属，来源 3708）：「这个包的吊牌之后是可以做个性化压印的，很多顾客会刻字母或者有纪念意义的日期，会更像一只专属的包包」
- 描述词疑问应对（来源 3296）：「很开心您对我们皮革类型描述的关注！这次皮革类型描述的更新是为了统一我们在不同产品中的用词，让顾客更清晰地了解我们所使用的皮革类型。请放心，我们在任何生产批次中使用的原材料始终保持一致，品质没有任何改变。这正体现了 Coach 对高品质皮革的坚持与专业承诺」
- 趋势 / 经典叙事素材（来源 3671，品牌 / 媒体叙事原文，非成交话术，引用须自行衔接顾客语境）：「本季我们希望展现所有让单品成为 Coach 经典的特质，这也体现在我们对材质的运用与其本质之美的呈现」（STUART VEVERS）；「通过高度耗时的工艺制作，每一款编织手袋、连衣裙与外套都展现出真正的奢华质感」
- 品牌理念叙事（来源 2047，面向学员的理念表述，落地为顾客话术须结合具体语境，勿机械复读）：「真我新奢（Expressive Luxury），是我们对奢华的接续发展的想象……更是关于自我表达的精神——拥抱各种面向的自己」；「Coach 的品牌宗旨是激励顾客自信探索自己的各个面向——拥有真实做自己的勇气」

**调性 / 结构提示（非话术原文，演练时以 KB2 真实产品填充）：**
- 价格三明治结构（2189）：【产品名称】→【产品价格】→【产品工艺及特色，重点强调价值、特色与优势】；价格被价值"包裹"，别裸报价。
- 经典传承句式（业务老师骨架）：把单品接回"设计灵感不过时 / 经典传承"，而不是"这个很好卖 / 大家都买 / 网红爆款"。
- 个性化认同（2189 进阶 + 业务老师骨架）：先听懂她的生活方式与需求，再挑最打动她的那个价值点，最后落一句真诚的"这款很适合您"，而非"很好卖"。
- GenZ 调性：像朋友分享、共鸣式、讲故事，不端着、不官方客服腔、不说教（2047 赋能式沟通 + 3739/3756 GenZ 调性）。

## 培训重点环节建议
| 环节 | 建议权重 | 训练重点 |
|------|----------|----------|
| 经典传承 / 故事加持 | 高 | 能把单品接回品牌叙事（经典传承 / 设计灵感不过时 / 本季趋势 Coach 诠释），用工艺与传统勾起情绪共鸣，而非平铺报参数 |
| BFA 把产品讲成价值 | 高 | 能否用 Benefits/Features/Advantages 把产品特征翻译成对这位顾客的好处与优势（而非干报特征）；不与 CONNECT 的 FABS 混淆（The Coach Experience p16） |
| 价格三明治（报价中） | 高 | 三层结构齐全（名称→价格→工艺特色），把价格夹住、重心放价值，不裸报价、不纠缠折扣；能把 BFA 好处填进三明治第①③层（POC 反馈"打动不了顾客"高频失分点） |
| 服务权益自然植入 | 中 | 在价值传递中自然带出免费刻字 / 护理 / 保修，强化专属与长期陪伴；口径不夸大（一年保修含拉链五金、护理仅清洁基础护理） |
| 个性化"适合您" | 高 | 按画像定制价值点，说"适合您"而非"很好卖"；落到 Expressive Luxury 自我表达，不把奢华窄化为"贵 / 大牌" |
| 价值 / 品质存疑异议应对 | 中 | 描述词更新口径（不得暗示降质）、解构 / 手工痕迹正向归因（不说成瑕疵）、皮质溢价给专业理由 |

「顾客在触感体验后停在一款经典系列包前，已经试背照过镜，但迟迟没说要不要。她心里在掂量'这个价到底值不值''是不是只是因为牌子贵'，也不想被当成只看 logo 的人。此刻轮到你做价值传递：先把这款接回品牌的经典传承与工艺（让她感到设计灵感不过时、不是一时爆款），报价时用价格三明治把价格夹在工艺特色之间、重心放在价值；再自然带出免费刻字与门店免费护理（让她感到买回去一直有人照顾、还能做成专属的一只）；最后不是说'这款很好卖'，而是结合她的生活方式与审美，真诚地告诉她'这款很适合您'。她要的不是一句'大牌就是贵'，而是一个能说服自己、且觉得'这是为我而选'的理由。」

## 在陪练系统中的应用
### 陪练对话引用
- AI 顾客在触感体验后进入价值传递场景，据"客户状态描述"表现出价格存疑、不想被当成只看 logo 的人、怕过时等心理。
- 对学员的价值传递作出差异化反应：当学员用经典传承 / 工艺渲染勾起情绪、用价格三明治把价格夹住、自然植入服务权益、并说出"适合您"的个性化理由时，AI 顾客情绪被打动、价值认同上升、价格戒备下降、愿意推进；当学员只说"大牌就是贵 / 很好卖 / 网红爆款"、裸报价纠缠折扣、含糊承认"可能没做好"、或暗示皮革"换了材料 / 降了品质"时，AI 顾客冷淡、把注意力锁回价格数字、产生品质 / 正品焦虑（详见 KB4-质感自用型"被什么话术推开"）。
- 触发本阶段异议：当报价或价值不被认同时，AI 顾客抛出"报价中"异议（价格 / 价值存疑、皮质溢价、描述词变化、解构像瑕疵、怕过时），考察学员是否"不谈便宜谈值"。

### 对话评估引用
作为"专业性：知识运用与价值传递"维度（POC 评估骨架，权重 30）及价值传递环节的判断依据：
- 是否完成经典传承 / 故事加持（讲品牌传统、工艺、经典 / 趋势，勾起情绪共鸣，而非平铺报参数）。
- 价格三明治是否运用到位（三层结构齐全、价格被夹住、重心在价值而非折扣）。
- 是否自然植入服务权益且口径不夸大（刻字 / 护理 / 一年保修范围准确）。
- 是否做个性化认同（按画像定制价值点，说"适合您"而非"很好卖"；落到 Expressive Luxury 自我表达，不把奢华窄化为"贵 / 大牌"）。
- 价值 / 品质存疑异议处理是否合规（描述词不暗示降质、解构 / 手工痕迹不说成瑕疵、皮质溢价给专业理由）。
- 沟通风格是否贴 GenZ、平等、带朋友感，不端着、不官方客服腔。
