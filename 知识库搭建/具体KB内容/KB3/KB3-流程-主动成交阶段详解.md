## 概述
**所属四阶段：CLOSE（成交）** —— The Coach Experience 四阶段 CONNECT → BUILD → CLOSE → CONTINUE 中，CLOSE 对应本项目"主动成交"（促成与连带），本篇即 CLOSE 阶段详解。

主动成交是 Coach 线下正价店 6 阶段销售流程的第 5 阶段（迎宾破冰 → 需求诊断 → 触感体验 → 价值传递 → **主动成交** → 售后维系），承接"价值传递"打动顾客之后的临门一脚。核心目标是：在顾客被打动、产生购买意向时，准确识别购买信号、用假设成交法复述确认、做场景化连带、并在缺货时把顾客稳稳接住——把意向转化为成交，同时不靠催促硬逼。本阶段是陪练系统"流程意识：识别购买信号、适时促成"维度的核心考点，也是 AI 顾客"被坚定背书后顺势答应 / 被硬推后退缩"两类反应的触发阶段。

## 阶段 / 场景定义
- **所属流程**：线下正价店 6 阶段销售流程（第 5 阶段）
- **适配渠道**：线下正价店为主；缺货承接（并行选择法）部分以品牌奥莱为课程语境，是否含正价店全渠道通用待确认（见来源与待确认事项）
- **所处阶段**：主动成交（促成与连带）
- **核心目标**：识别购买信号 → 复述确认锁定 → 场景化连带 → 稳妥处理缺货与临门异议 → 完成成交（业务老师骨架定位为"促成与连带"）
- **销售强度**：高（成交临界点，但 Coach 口径是"坚定又让顾客被理解"，非施压硬逼）

## 客户状态描述
进入本阶段的顾客通常已经过触感体验和价值传递，对某一两款产品产生了明确兴趣，处于"想买但还差一个被坚定背书的理由"的成交临界状态。典型表现（来自 2191 课程购买信号清单）：
- **非语言信号**：持续的眼神交流并展露微笑；长时间拿着产品、在镜子前面观赏（把产品代入自身使用场景，在想象拥有它的样子）。
- **语言信号**：询问关于售后服务 / VIP 制度（开始关心买后保障与会员身份，是积极信号而非异议）；询问你或别人的意见（寻求决策背书，想被确认这个选择是对的、值得）。

同时，本阶段顾客也常带着临门一脚的犹豫与戒备：
- **"两款都喜欢、难取舍"** 的决策焦虑（怕选错、买贵、不值）。
- **缺货失望**：带着明确目标（社媒同款 / 当季流行色 / 草编蝴蝶结等）进店却扑空，失望但未必明说，需要被"接住"而非被劝退。
- **报价后的价格/比价/拖延异议**：对应业务老师"报价后（成交推进）"环节——"有点贵 / 再看看 / 问家人"，需判异议真伪再选策略。
- **信任戒备**：怀疑导购是不是只想把东西卖掉（"你是不是只想要把这个东西卖掉才鼓励我买？"）。

> 不同行为类型在本阶段戒备点不同：质感自用型反感过度推销、爱自己判断；性价比入门型纠结"花一千多会不会太奢侈/同事觉得乱花钱/能背几年不过时"；礼品选购型反复确认"够不够正式、对方会不会喜欢"；都市悦己型决策高效、不轻易接受替代、确认有货即高效成交。

## 标准流程
> 业务老师骨架原文（A1·主动成交）："假设成交法复述确认(商品/颜色/刻字)→场景化连带(包+同色卡包,组合价)→支付引导含权益告知。" 课程在此骨架上补足识别信号（E.N.D.）、缺货承接（并行选择法）与异议克服（A.A.A.）的具体动作。

1. **识别购买信号（启动成交的判断动作）**：观察顾客是否发出 4 类购买信号（眼神+微笑 / 询问售后VIP / 长时间持物镜前观赏 / 征询意见）。出现任一或多个，即判断兴趣已达成交临界，抓住"在当下"的时机切入成交，不让热度冷却。详见 KB3-方法-E.N.D.购买信号识别与成交。

2. **运用 E.N.D. 协助犹豫顾客做决定**：当顾客在两款（或多款）之间犹豫时，按固定顺序推进——**E 排除**（缩小选择范围，帮顾客排除不合适那款并说明理由）→ **N 需求**（重申顾客分享过的功能性 + 情感性需求，强调所选单品如何满足）→ **D 决定**（强而有力的专业陈述坚定其想法，并明确请求购买，如"来，我帮你包起来吧"）。态度坚定，同时让顾客感觉被听见、被理解。

3. **假设成交法复述确认**（业务老师骨架核心动作）：在顾客意向明确时，以"已经要买了"的口吻复述并确认关键决策项——**商品 / 颜色 / 刻字**（例如确认是哪一款、哪个颜色、是否需要免费刻字及刻什么）。复述确认既锁定成交，也给顾客二次确认的安全感。注：免费刻字属增值服务权益，具体权益以 KB1 增值服务体系 / 门店主数据为准。

4. **场景化连带（促成与连带）**（业务老师骨架核心动作）：在主商品确认后，做场景化的附加销售——典型组合是"包 + 同色卡包（小皮具/包挂），按组合价"。连带要落到使用场景而非堆叠推销（如"这只包配同色卡包，整套出门更统一"；春季书本挂饰可按书目主题/搭配做情感化连带）。报价附加品时同样可用价格三明治，降低加购的价格敏感（价格三明治"建议场景"明确含"推荐附加销售产品的时候"）。**组合价 / 折扣具体数字课程未给 → 以门店主数据为准，勿编造。**

5. **缺货时用并行选择法承接（不硬逼、不将就）**：若顾客想要的颜色/款式/社媒同款已售罄、断货，按 A.U.P. 三步承接——**确认 Acknowledge**（先肯定顾客眼光与选择，不急于转移）→ **理解 Understand**（说出顾客"真正想要的"感觉/场景，而非锁死某个 SKU）→ **呈现 Present**（给"同样适合"的并行选项，不替代、不施压），帮顾客确认"我选另一个，也很对"。店内确实缺货时可提及 Link to EC / Omni Hub 等全渠道承接路径。详见 KB3-方法-并行选择法与缺货承接。

6. **临门异议用 A.A.A. 化解（不硬逼）**：报价后顾客出现"有点贵 / 再看看 / 问家人 / 怕以后后悔 / 你是不是只想卖掉"等异议时，按 **Acknowledge 回应 → Ask 询问 → Address 处理解决** 三步，"先共情后解决"，并主动了解顾客未表达的潜台词。绝不打断、纠正、预判顾客想法或只共情不给方案。详见 KB3-方法-A.A.A.异议处理技巧。

7. **支付引导含权益告知**（业务老师骨架收尾动作）：在顾客决定购买后引导支付，并自然告知相关权益（如免费刻字、皮具护理等增值服务）。权益内容以 KB1 增值服务体系 / 门店主数据为准，勿编造 VIP 权益或退换条款。

## 该阶段典型异议及应对
> 承接 KB4 异议总表"报价前/中/后"分布：本阶段对应 **报价后（成交推进）**——"有点贵 / 再看看 / 问家人"，需判异议真伪再选策略；缺货同款焦虑、决策犹豫亦集中在本阶段。仅写关切主题、潜台词与应对方向，不写剧本台词。

| 关切主题 | 顾客潜台词 | 应对方向 | 关联行为类型 |
|----------|------------|----------|--------------|
| 报价后价格存疑 | 有点贵 / 不确定值不值这个价 | 判异议真伪；用 A.A.A. 承接，不谈便宜谈值，配合价格三明治把价格夹在产品与工艺特色之间，回到品质-服务-使用频率三维撑价 | 性价比入门型 / 多类型通用 |
| 两款都喜欢、难取舍 | 怕选错、买贵、不值，想被帮着做决定 | 用 E.N.D.：先 Eliminate 缩小范围、再 Needs 重申功能+情感需求、最后 Decision 强陈述+请求购买 | 多类型通用 |
| 拖延（再想想/再看看） | 现在做不了决定、想清楚再买 | 尊重节奏不催促（"想清楚再买是对的"）；挖背后吸引点；缺货款坦白不补、用在售款承接；留联系方式下次再陪看 | 性价比入门型 / 都市悦己型 |
| 需他人背书（问家人/朋友） | 想让家人朋友看了再决定 | 帮带人再来 + 留货减负担；尊重决策节奏，不施压 | 礼品选购型 / 性价比入门型 |
| 想要的款/色已缺货 | 我想要小红书同款 / 粉绿浅色 / 草编蝴蝶结，怎么没有 | 并行选择法 A.U.P.：肯定眼光→说出真正想要的感觉→给同样适合的并行选项；禁说"这个没了要不要看别的""只能换这一款" | 都市悦己型 / 性价比入门型 |
| 颜色与预期不符 | 怕实物色差、选错色不甘心 | A.A.A. 回应色差关注→询问期待的色感场景→对比/搭配/真实呈现 | 都市悦己型 / 质感自用型 |
| 未来后悔焦虑 | 怕冲动消费、长期不喜欢，隐含退换与保值担心 | A.A.A. 承接焦虑→询问犹豫点→用经典长期主义/质感投资重构 | 性价比入门型 / 质感自用型 |
| 怀疑导购推销动机 | 你是不是只想把东西卖掉才鼓励我买 | 不辩解不施压，重申以顾客需求为中心、用心找最适合的产品 | 多类型通用 |

## 关键动作与话术参考
> 口语化、贴 GenZ 风格。话术原文来自课程，已标来源；陪练 AI 顾客侧异议请用"关切主题+潜台词"，不机械复读整句。

**识别购买信号 + E.N.D. 成交（2112/2191，教学示例话术，示例产品参数非真实在售）**
- 【DECISION·请求购买】"这真是一款能满足你所有需求的完美手袋。你背起来也超好看……来，我帮你包起来吧。"
- 要点：识别到"长时间拿着包、镜前观赏 / 问售后VIP / 征询意见"任一信号，就别再平铺介绍，顺势切入成交。

**假设成交法复述确认（业务老师骨架）**
- 复述确认结构（结合 E.N.D. 的 D 步）："那我们就定这只 XX、XX 色，要不要顺便刻上你的名字首字母？刻字是免费的。"（商品 / 颜色 / 刻字三项一次确认；权益以门店主数据为准）

**场景化连带（业务老师骨架 + 价格三明治"附加销售"场景）**
- 落到场景而非堆推："这只包配一只同色卡包，整套出门更统一，组合价更划算。"（组合价数字以门店主数据为准）

**缺货并行选择法 A.U.P.（3697 课程原文）**
- 【确认 Acknowledge】"这个颜色/款式，确实是现在很多人关注的款，你的眼光很好！"
- 【理解 Understand】"你会选这个，其实是想要轻松、好搭，很有夏天的感觉。"
- 【呈现 Present】"其实真的不只一种选择适合你，我们可以一起看看，哪一个更适合你、你更喜欢。"
- 【缺货承接锚点】"我选另一个，也很对。"（把缺货变成一次专业判断的目标句）
- 【再想想/等补货】"没关系，真的想清楚再买是对的。"（不催促，把决定权还给顾客）

**临门异议 A.A.A.（2190，先共情后解决）**
- 顾客异议开口素材（顾客台词，供 AI 顾客参考，非导购话术）："如果我以后讨厌这个产品怎么办？""你是不是只想要把这个东西卖掉才鼓励我买？"
- 导购应对方向：先回应承接情绪 → 开放式询问真实顾虑 → 给有效方案；面对推销动机质疑时重申"以你的需求为中心，帮你找最适合的"。

**禁语红线（缺货承接，3697 明确列出）**
- 禁说："这个已经没了，要不要看看别的？""那只能换这一款了。"——把缺货说成"将就"会把顾客推开。

## 培训重点环节建议
> 权重为训练侧建议，非客户确认的固定评分权重；评分权重以 KB5-评分-评分规范为准。

| 环节 | 建议权重 | 训练重点 |
|------|----------|----------|
| 识别购买信号 | 高 | 能否准确读出 4 类信号、不把"问售后VIP"误判为异议、抓住"在当下"时机切入 |
| E.N.D. 三步执行 | 高 | Eliminate 缩范围 / Needs 兼顾功能+情感 / Decision 强陈述+明确请求购买，顺序不乱、态度坚定又让顾客被理解 |
| 假设成交法复述确认 | 中 | 商品/颜色/刻字三项一次锁定，给顾客二次确认的安全感 |
| 场景化连带 | 中 | 落到使用场景而非堆推；用价格三明治降低加购价格敏感；组合价不编造 |
| 缺货并行选择承接 | 高 | A.U.P. 三步、说出"真正想要的感觉"、避免禁语、把缺货转为专业判断 |
| 临门异议 A.A.A. | 高 | 先共情后解决，不打断/不预判/不只共情不解决，判异议真伪 |
| 不硬逼底线 | 中 | 尊重决策节奏、留货留联系，不施压成交（合规与品牌体感） |

## 该阶段典型描述模板
> 一段可直接用于陪练场景情境的描述文本：

"顾客已经在镜子前背着这只包看了好一会儿，眼神发亮还问了句'你们刻字收费吗'——这是明显的购买信号。她其实想买，但在这只和另一只之间有点犹豫，心里还闪过'花这么多会不会太冲动''要不要回去问问家人'。这时学员需要抓住当下的时机，用 E.N.D. 帮她排除一款、重申她在意的功能和情感需求、给出坚定又真诚的购买建议并主动请求成交；用假设成交法把商品、颜色、刻字一次确认下来；顺势做一个落到使用场景的连带（比如同色卡包组合）。如果她想要的颜色刚好缺货，要用并行选择法'确认眼光→说出她真正想要的感觉→给同样适合的选择'稳稳接住，而不是说'这个没了换一款吧'。全程坚定但不硬逼，让她觉得是被理解、被帮着做了一个对的决定。"

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客据本阶段流程对学员成交动作作出差异反应：
- 学员准确读出购买信号、用 E.N.D. 先排除→重申功能+情感需求→坚定请求购买（态度坚定又让顾客被理解）、并以假设成交法复述确认 → 顾客顺势点头、悠然答应；
- 学员错过信号继续平铺介绍、或只健谈不促成、或害羞不敢请求成交、或把决定丢回给顾客 → 顾客继续犹豫、热度冷却；
- 顾客想要的款/色缺货时，学员走通并行选择法 A.U.P.、避免禁语 → 顾客被"接住"、愿意看并行选项；学员说"这个没了换一款吧"或将就式推荐 → 顾客失望变冷淡甚至被推走；
- 学员面对价格/后悔/推销动机异议用 A.A.A. 先共情后解决、不硬逼 → 顾客戒备下降、信任回升；学员催促硬逼或只共情不解决 → 顾客抗拒加深。
AI 顾客的犹豫潜台词与缺货失望、决策焦虑素材来自 KB4。

> **陪练对话推进机制（非课程知识，供角色扮演链使用；来源：业务老师反馈 KB3-3）**
> 当学员在本阶段只被动回应 AI 顾客的提问、迟迟不主动推进成交时，**AI 顾客应在两三个对话回合内，根据学员此前的整体表现（前几阶段是否打动自己、是否建立信任），主动选择是否替学员把对话往下推进到成交阶段**——表现好则顺势配合、给出更强的购买意向信号甚至主动问"那怎么买"；表现不足则可保持观望、停在犹豫态，不替学员"送"成交。
> 这是**陪练对话的推进/接管机制**（解决"学员只会接话、不会促成"时对话卡死的问题），不是 Coach 课程里的销售知识，也不写入对客话术；它约束的是 AI 顾客的扮演行为，目的是让陪练对话能继续、并把"是否主动促成"这一能力暴露给评估。具体回合数与触发判定以场景编排/角色扮演链配置为准。

### 对话评估引用
作为"流程意识：识别购买信号、适时促成"与"主动成交"表现的判断依据：
- 是否识别并回应 ≥1 类购买信号、是否抓住"在当下"时机；
- E.N.D. 三步是否完整且顺序正确（Eliminate / Needs 兼顾功能+情感 / Decision 强陈述+明确请求购买）；
- 假设成交法是否复述确认商品/颜色/刻字；
- 连带是否落到使用场景、是否避免堆推与编造组合价；
- 缺货承接是否走通 A.U.P.、是否避免禁语、是否把缺货转为专业判断；
- 临门异议是否用 A.A.A. 先共情后解决、是否踩 4 条红线（预判/打断/不提问/不给方案）、是否判异议真伪；
- 是否守住"不硬逼"底线（尊重节奏、留货留联系、不施压成交）。
