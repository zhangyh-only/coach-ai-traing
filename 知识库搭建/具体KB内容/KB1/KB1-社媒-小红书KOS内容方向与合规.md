## 概述
本文档描述 Coach 线上社媒渠道（小红书 KOS / Key Opinion Sales）的内容运营方向与评论区引导合规红线，是 KOS 在小红书做种草内容、与顾客在评论区互动时的渠道背景与边界规范。它包含两块可被陪练系统引用的事实：一是月度社媒热点穿搭趋势词典（6 个趋势话题），二是评论区私域引流的 DON'T / DO 合规框架。前者决定 AI 顾客在社媒语境下的话题与情绪偏好，后者决定学员在线上评论区引导顾客时"可说 / 不宜说"的边界。本快报具有强时效性（2026 年 4 月当季话题），趋势内容须作为"季度话题"引用，不可当作长期产品事实。

## 核心内容

### 一、渠道定位：小红书 KOS 内容运营
- **渠道属性**：线上社媒渠道，由 China KOS Team 内容运营，KOS = Key Opinion Sales（社媒导购 / 种草角色）。非 Retail / Outlet 门店场景。
- **内容叙事**：以生活方式化、视觉驱动的"种草"叙事承载品牌理念 Expressive Luxury（真我新奢）——把 Coach 包袋嵌入"松弛、随性、真我表达"的穿搭与生活场景，而非硬卖产品参数。
- **内容结构（快报版面反复出现的标准结构，供识别 KOS 种草口径）**：
  1. 趋势标签（#话题）+ 一句情绪化副标题（如"不费力的松弛感""上下班无缝切换"）
  2. "什么是 X？"段落给风格定义
  3. "搭配技巧 / 穿搭技巧"列出可执行清单（面料 / 色系 / 单品 / 包袋）
  4. 一句口号金句收尾（如"流汗也要流得优雅""人在工位，心在巴厘岛"）
- **平台流量扶持**：#ColorWalk 被标注为"平台流量扶持活动"，有参与方式与流量 / 收益激励机制。具体激励数值（曝光、活动资源等）课程版面字号极小、辨识不确定 → 见"待确认事项"，**不臆测数字**。

### 二、月度社媒穿搭趋势词典（2026 年 4 月·强时效·季度话题）
> 以下 6 个趋势话题词为 2026 年 4 月当月热点，**原样保留不翻译**；引用时须标注当月时效，避免当作长期产品知识。课程仅给品类化提及，无具体在售 SKU / 价格 / 配色货号（**以门店 / 电商主数据为准**【待客户补充】）。各趋势"代表单品 / 搭配建议"的实拍海报小字部分辨识受限 → 见"待确认事项"。

| 趋势标签 | 风格定义（课程口径） | 代表单品 / 搭配要点 | 情绪 / 口号金句 |
|----------|----------------------|---------------------|-----------------|
| **#ColorWalk** | 平台流量扶持活动 + 色彩主题穿搭方向（亮色 / 撞色行走城市） | 课程以活动机制呈现，搭配细文案辨识受限【待客户补充】 | （活动话题）|
| **#Offduty** | 上下班无缝切换的松弛得体——"不会让你看起来像刚下班去开会，也不会像周末在家躺平" | 面料软但挺括（软牛仔 / 棉麻 / 府绸 / 微弹针织）；色系干净（黑白灰 / 卡其 / 海军蓝 / 森林绿 + 一点亮色）；正式休闲混搭；**包袋随性但不随便**：Coach 斜挎包 / 腋下包 / Hobo 包，亮色可选焦糖 / 酒红 / 森林绿 | "上下班无缝切换" |
| **#Softfit** | 追求极致舒适与内在力量、回归自我的不费力时髦感；松弛、安静且高级 | 面料选软（针织 / 缎面 / 软牛仔 / 褶皱）；反堆叠求松弛（不贴身、肩线柔和、留呼吸感）；色系温柔（奶茶 / 奶灰 / 奶白、低饱和）；叠搭有层次；点睛配饰选一只软皮包、颜色亮一点 | "不费力的松弛感" |
| **Sportique** | 运动场地的随性注入诙谐精致气质（运动风 × 精致） | 运动背心 + 微喇裤 + 薄底鞋；运动连衣裙 + 长筒袜 + 复古跑鞋 | "流汗也要流得优雅" |
| **叛逆学院风** | 学院风的锋利 / 态度表达——"乖张学姐放学别走，皮革斯文外表下的锋利灵魂" | 撞色针织 + 百褶裙 + 厚底乐福；皮革短外套 + 制服西裤 | "乖张学姐放学别走，皮革斯文外表下的锋利灵魂" 【待业务确认删除：此条为非口语化的海报标语式文案，业务老师建议删除以免陪练对话显死板】 |
| **Staycation** | 把度假感穿进日常生活，都市快节奏下的情绪代偿 | 流苏上衣 + 休闲裤 + 平底鞋；镂空 / 草编元素（草编包） | "人在工位，心在巴厘岛" 【待业务确认删除：此条为非口语化的海报标语式文案，业务老师建议删除以免陪练对话显死板】 |

- **Coach 包袋的社媒口径**：在 #Offduty / #Softfit 等松弛场景中，把 Coach 斜挎包 / 腋下包 / Hobo 包 + 亮色（焦糖 / 酒红 / 森林绿）嵌入"随手一背就是下班状态"的生活方式叙事，做软性种草，**不脱离场景硬推、不编造款号 / 参数**。

### 三、评论区引导合规红线（DON'T / DO）—— 本课核心
> 约束 KOS 在小红书评论区承接顾客、做私域引流时的合规边界。平台对引流行为有系统识别 / 拦截机制；红线一律禁止，合规出口只有"服务型话术 + 表情 + 引导至主页"。话术原文随附（来源：3698 快报评论区示范，原样保留；部分 emoji 渲染受限为近似还原）。

**DON'T（禁止项·私域引流红线）**

| 禁止类别 | 禁止行为 | 平台处置（课程标注） |
|----------|----------|----------------------|
| 直接引导类 | "加我微信细聊""点击主页进群" | 系统识别**高风险** |
| 联系方式类 | 留微信号 / 手机号 / 二维码；🔗外链 / 链接 | 系统**直接拦截** |
| 诱导互动类 | "私信才发你" | 禁止 |
| 谐音规避类 | "厚台哦 / 滴滴"等谐音绕过审核 | 同判高风险，禁止 |

**DO（推荐项·合规做法）**

| 推荐做法 | 合规话术示范（原文，含落款来源） |
|----------|----------------------------------|
| 服务型话术，用表情代替直接引导 | 「大家有任何问题可以随时来找我👋♀️」（评论示例落款：BV 成都 SKP Kitty / 作者 / 2024-11-29 回复）|
| 引导至主页、用表情谐音代替"关注" | 「这个博主主页太精彩了，果断 📱🐯🐯」（评论示例落款：一颗黄心 / 2025-11-30 北京 / 72 赞）|

- **私域引流边界小结**：评论区**不直白索要联系方式、不直接加微 / 留号、不外链、不诱导、不用谐音绕审核**；想提升关注 / 主页流量时，唯一合规出口是"引导至主页 + 表情谐音代替'关注' + 服务型话术"。

## 在陪练系统中的应用
### 陪练对话引用
- 当陪练场景设定为线上社媒 / 小红书种草语境时，AI 顾客（偏都市悦己型 / 质感自用型）据本文档的趋势词典表达话题与情绪偏好：追求"松弛感 / 不费力""把度假感穿进日常""用穿搭做态度表达"，对标签化、情绪金句、亮色点缀、生活方式化种草有正反馈。
- 当 AI 顾客在评论区提出私聊 / 加好友诉求时，本文档为学员"该如何合规承接"提供边界：学员若直接索要微信 / 留号 / 外链 / 谐音绕审核，AI 顾客（与平台逻辑）会表现出被硬性私域引流的警惕 / 反感而被"推开"。

### 对话评估引用
- 渠道合规 / 沟通禁忌红线：评判学员在线上评论区是否触碰 DON'T（直接加微、留联系方式、外链、诱导互动、谐音规避），是否用合规 DO（服务型话术 + 表情 + 引导至主页）承接。
- 时效性 / 内容准确性：评判学员引用社媒趋势时是否说明为当季（2026 年 4 月）热点、避免把月度时效话题当成永久产品知识。
- 价值传递 / 产品知识准确性：评判学员是否把 Coach 包袋嵌入生活方式场景做软性种草，而非脱离场景硬推或编造款号 / 参数。
