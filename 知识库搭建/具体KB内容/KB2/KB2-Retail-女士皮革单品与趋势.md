## 产品详细介绍

### 设计灵感与结构

**本期趋势主推：水桶包（Bucket Bag）**（来源：3737「趋势特报：BUCKET BAG」）
- 月度「趋势特报」机制框定的当季流行包型，是 6 月 Retail 女士皮革单品的核心趋势单品。
- 视觉可见两种材质呈现：
  1. **光面皮革款**：黑色光面皮，金色气眼（grommet）穿肩带、抽绳收口、正面 COACH 烫金、挂金色字母吊坠。
  2. **signature 涂层帆布抽绳款**：黑色双 C 字母涂层帆布、金色五金、抽绳结构。
- 结构特征：气眼 + 抽绳收口；背法多样，街拍中出现斜挎、手提、单肩等多种背法。

**满铆钉中号皮革手提包**（来源：3767「材质基础」）
- 结构特征：包身满铺铆钉装饰（边角/底部）、正面贴袋、Coach 旋转锁扣（Turnlock）、双圆角提手、皮质挂牌（Hang Tag，压印 COACH）、马蹄形/字母吊饰挂坠、可拆卸长肩带（带龙虾扣）、车缝明线。
- 同款多材质呈现：焦糖棕光面皮、黑色光面皮、棕色绒面/麂皮质感。

**其他视觉可辨识款式族**（来源：3737，造型/趋势示范）
- Tabby 系列肩背款（signature C 衔扣五金、皮牌）、Tabby Pillow/软身款（signature C 五金，搭 C-logo 印花服装）。
- 皮革 Hobo/单肩大包，配 crochet 钩织小熊 + 樱桃 + 骰子挂饰（GenZ 个性化配件）。
- 马衔扣五金翻盖单肩包（runway/秀场造型示范；是否为在售款【待客户补充】）。

### 核心卖点

1. **当季趋势属性（水桶包）**：本期被框定为流行包型，对应"想拥有当季流行款"的潮流心理。介绍时点出其当季流行属性。
2. **材质质感双选**：同款常呈现光面皮（油蜡光泽、平滑）与绒面/麂皮（柔软磨砂手感）两种质感，给顾客"按个人风格选质感"的自我表达空间——天然的触感体验素材。
3. **标志性设计辨识度**：满铆钉、Coach 旋转锁扣（Turnlock）、皮质挂牌（Hang Tag）、字母 C 吊饰、signature C 五金等构成强品牌辨识，契合"被认出是 Coach"的悦己/表达心理（Expressive Luxury 真我新奢）。
4. **一包多背、多场景适配**：水桶包街拍覆盖牛仔休闲、格纹大衣通勤、白西装正式、秀场造型，可按顾客生活场景演示不同背法与搭配。
5. **可个性化（挂饰）**：crochet 钩织小熊/樱桃/骰子等挂饰与 C-logo 服装，承载"独一无二、可玩可搭"的自我表达（与 Expressive Luxury 一致）。挂饰是否为在售配件及其参数【待客户补充】。

### 客户常见问题（FAQ）
> 两课均无烤入 Q&A/测验题，以下为基于视觉事实可合理沉淀的高频问答方向；具体参数答复一律以门店主数据为准，不臆造。

**Q1**：这个水桶包是今年流行的款吗？
**A1**：是本期趋势特报主推的当季流行包型；视觉上有光面皮革款和 signature 涂层帆布款两种材质可选。（具体款名/价格/容量以门店主数据为准）

**Q2**：水桶包能怎么背？平时好搭吗？
**A2**：可斜挎、手提、单肩多种背法；街拍里搭牛仔休闲、格纹通勤、白西装正式都有，属于一包多场景适配。

**Q3**：这只包是什么皮？光面的和那只磨砂的有什么不一样？
**A3**：视觉上一种是光面皮（油蜡光泽、平滑），一种是绒面/麂皮质感（柔软磨砂手感），可以摸一下对比手感。（皮革官方品名以门店产品主数据为准，不臆造材质学名）

**Q4**：这个包能装多少东西？
**A4**：以同款黑色手提包的场景为参考，可容纳手机、墨镜盒、笔记本、笔等日常通勤物品。（具体容量数据以门店主数据为准）

**Q5**：包上这些钩织小熊/樱桃挂饰是 Coach 的吗？能单独买吗？
**A5**：挂饰是个性化搭配的卖点之一，体现可玩可搭的自我表达。（是否为在售配件、款名与价格以门店主数据为准）
