## 产品详细介绍

### 一、品类体系（按渠道与场景）
Coach 在售品类以女士皮革包袋为核心，向外延伸到小皮具、男士斜挎、鞋履与成衣。三门课程可见的品类骨架如下：

- **女士皮革包袋（核心）**：肩背款、软身/枕形款、Hobo 单肩大包、翻盖单肩包、水桶包、托特、斜挎/相机包等。Retail 与 Outlet 均以此为主力。
- **男士斜挎补充款**：Axel、Finlay、Heritage、Fletcher 等斜挎包（Outlet 课程作为 Colton 的男士关联热卖款出现）。
- **小皮具**：卡包/钱包类（如 Outlet 上货表中的 MINI ID SKINNY 等，属货品规划口径）。
- **鞋履 FTW 与成衣 RTW**：Rexy 10th Anniversary 等周年企划（FTW 鞋履 / RTW 服饰）。

> 说明：以上为三门课程出现过的品类线索汇总，非完整在售品类清单。各品类完整 SKU、价格、容量以门店主数据为准。

### 二、正价 vs 奥莱：同一个人在两家门店会问出的不同问题【本文核心】
顾客带进门的关切，本身就因门店而异。同样想买包的人，**走进正价店和走进奥莱，第一句话往往就不一样**——与其背一张皮质参数对照表，不如让陪练 AI 顾客记住"这两类店里，人会更在意什么、更可能问什么"。下面以"问题视角"组织（问题来源于课程渠道事实推得的顾客高频关切）：

| | 正价店(Retail) 里顾客更可能问 | 奥莱(Outlet) 里顾客更可能问 |
|------|--------|--------|
| 开场关切点 | 这是不是新款 / 当季在主推什么？ | 这个为什么便宜 / 是不是正品？ |
| 典型提问 | "有没有新到的款""这个是经典款还是限定""这皮是什么材质、工艺怎么样" | "现在打几折""奥莱的是不是材质差、缩水""为什么比正价店便宜" |
| 在意的事 | 新意、经典/限定身份、材质工艺质感、值不值得长期持有 | 折扣力度、性价比、正品与否、这个价钱买得划不划算 |
| 心理底色 | 质感投资 / 自我犒赏，看重"独特和品质" | 精打细算 / 怕被坑，看重"便宜但不能踩雷" |

> 用法：陪练时 AI 顾客根据"现在在哪类门店"调取这一栏的提问倾向，自然开口；学员要能听出顾客真正在意的是"新意/质感"还是"折扣/正品"，对症回应，而不是把两边当成同一种顾客。

**回应"奥莱是不是材质差/不是正品"的事实底座（产品线差异，非真假之别）**：
- Retail 与 Outlet 同属 Coach 官方渠道。差异在产品线：Retail 除牛羊皮外**还有蛇皮、鳄鱼皮等稀有皮**，多用**头层皮**，鞣制工序更少、更还原天然纹理，皮面修整更丰富；Outlet 主要为**牛、羊皮**，**大部分使用分层皮**（Outlet 侧的鞣制/修整工艺课程未展开，以门店主数据为准）。
- Signature C 涂层帆布也有产品线差异：Retail 双 C 较小、深底浅字、随机一个 C 内暗藏 Coach 缩写标签；Outlet 浅底深字、无额外暗签——这是设计差异，**不能反推 Outlet"非正品"**。
- **红线**：客观说明差异，绝不贬低 Outlet（不说"次品/假货/缩水"），也不夸大 Retail。完整正品/防伪政策与服务边界以门店 SOP 为准。

> 注意：以上产品线事实来自 3365 皮革知识课；3697（Outlet 新品课）与 3737（Retail 趋势课）均未自行做 Retail vs Outlet 皮质对比，渠道差异事实统一以 3365 为准。

### 三、渠道运营逻辑差异（价格逻辑 / 库存 / 承接）
- **Retail 正价店**：以头层皮、稀有皮与更丰富皮面修整承托质感与价格；月度趋势特报引导当季单品（视觉种草 + 多场景搭配）。价格逻辑落在"质感投资/正品保障"叙事，具体价格以门店主数据为准。
- **Outlet 奥莱**：季节新品上新（如出游档 Newness），主打流行色(浅粉/肉桂棕/石板灰)、热门面料(丹宁/花卉/条纹 Signature 帆布)与高性价比包型。**库存承接是 Outlet 特有逻辑**：当季热门色/款常 2-3 个月不补货，缺货时用"创造选择而非创造库存"的并行选择法承接（详见 KB3-方法-并行选择法）。库存/上货表为内部货品规划数据，**非对顾客话术内容**。

### 客户常见问题（FAQ）
> 总览级 FAQ，聚焦"渠道/品类归属"类问题；具体材质应对话术见 KB3-方法-皮革材质异议应对话术。

**Q1**：Retail 和 Outlet 的包有什么区别？奥莱的是不是材质差/不是正品？
**A1**：两边都是 Coach 官方渠道，区别在产品线：Retail 除牛羊皮外还有蛇皮、鳄鱼皮等稀有皮，用头层皮、鞣制工序更少更还原天然纹理、皮面修整更丰富；Outlet 主要牛羊皮、多用分层皮。这是产品线差异，不是真假之别——客观说明，不贬低 Outlet。（事实底座见上"正价vs奥莱差异"表）

**Q2**：你们最近主推什么包？
**A2**：分渠道看——Retail 走月度趋势特报（如 2026 年 6 月主推水桶包，光面皮与 signature 帆布两种材质、一包多背）；Outlet 走季节新品上新（如出游档的浅粉/肉桂棕/石板灰三色与丹宁/花卉/条纹帆布面料）。具体在售以门店主数据为准。

**Q3**：这个包多少钱 / 什么款名 / 容量多大？
**A3**：三门课程均未给出价格，部分容量有给（如 Rainer 装 Pro max、Elodie 装 11 寸 iPad）。凡课程未明文的款名/价格/容量，一律以门店主数据为准，不臆造。

### 常见异议及应对（关切主题，不写台词）
| 关切主题 | 应对方向 |
|----------|----------|
| 奥莱皮质是否缩水/是否正品 | 客观陈述 Retail/Outlet 产品线差异，强调同为官方渠道、非真假之别，不贬低 Outlet |
| 当季主推是什么、流不流行 | 按渠道分别给趋势特报(Retail)或季节新品(Outlet)主线，落到具体系列定位 |
| 款名/价格/容量追问 | 明确以门店主数据为准，已知容量(Rainer/Elodie)可给，未知不臆造 |
| 流行款会不会过时/值不值 | 引导到具体产品篇的趋势/经典叙事；经典款(Tabby)讲长期主义，趋势款讲当季适配 |

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客先据"自己现在在哪类门店"调取第二节"问题视角"那一栏：在正价店更会从新款/经典款/材质工艺切入，在奥莱更会从折扣/正品/为什么便宜切入，由此自然开口、贴合该渠道语境。对款名/价格类追问，AI 顾客据本文"以门店主数据为准"原则，识别学员是否在臆造参数。本文是入口，具体材质/卖点话术下钻到各产品篇与 KB3。

### 对话评估引用
作为"渠道与品类归属判断是否正确"的评判依据：学员是否听出顾客真正在意的是"新意/质感"（正价语境）还是"折扣/正品"（奥莱语境）并对症回应、是否客观说明 Retail/Outlet 差异而不贬低 Outlet、是否把顾客带到正确的产品线、是否避免编造课程未给的款名/价格/容量。细颗粒的材质准确性评估见 KB5-评估-产品与材质说明准确性。
