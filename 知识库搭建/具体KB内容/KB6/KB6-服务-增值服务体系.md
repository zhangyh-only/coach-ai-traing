# 增值服务体系

文档编号：KB6-服务-增值服务体系

## 概述
本文档汇总 Coach 线下正价店（Retail）成交后可主动介绍的增值服务边界，覆盖企业微信跟进、全国门店一年保修与退换、皮革护理（清洁与基础护理）、个性化压印、搭配建议延展五类服务。定位是"售后维系/成交后体验"环节把后台系统操作流程翻译为顾客可感知的线下尊享体验的参考材料。供陪练 AI 顾客判断"买完之后有没有人管我"，供评估判断学员是否主动讲全增值服务、是否准确说明服务边界且不夸大、不编造。

## 核心内容

### 1. 添加企业微信（售后跟进入口）
- **权益内容**：成交后第一步引导顾客扫码添加导购本人企业微信，作为长期触达入口，承载下单出示会员码/填写物流信息、售后跟进、搭配建议推送、护理提醒、个性化压印对接等持续服务。
- **适用条件**：成交/交付环节主动引导；线上下单到店履约同样可用企微跟进。
- **如何在对话中自然介绍**：以"我"为责任主体把企微定位成"找我跟进售后,更快更方便"的专属顾问入口，而非生硬要求加好友。参考话术见 KB3-售后维系阶段（来源 3708）。

### 2. 售后维修与退换货（全国门店一年保修）
- **权益内容**：全国门店提供一年保修；课程举例保修范围"包括拉链、五金配件等"；线上下单的产品全国门店同样提供一年保修；礼赠场景下"送人对象觉得不合适,可到店或直接找本人处理"。
- **适用条件 / 边界**：
  - 保修期限=一年；范围=门店举例"拉链、五金配件等"，"等"字之外是否覆盖皮面、缝线、人为损坏，课程未说明 → 【待客户补充】（需门店售后 SOP 确认），导购不宜夸大为"全包/终身保修"。
  - 线上下单同样享全国门店保修，不可称"线上买的不享门店售后"。
  - 退换货的具体条件、时效、是否收费、线上线下差异，课程未给出 → 【待客户补充】（需退换政策 SOP），导购不宜编造承诺。
- **如何在对话中自然介绍**：保修举例据实说"拉链、五金配件等"，礼赠顾虑用"不合适可到店或找我处理"承接，不甩政策、不夸大。

### 3. Coach 皮革护理服务（免费清洁与基础护理）
- **权益内容**：
  - 包可带到任意一家 Coach 门店，门店提供免费清洁和基础护理（3708：用一段时间后"都可以免费帮您做清洁和基础护理,相当于一直有人帮您照顾它"）。
  - 护理服务里提供防水喷雾程序，"能有几率做到轻微防水"（3365）。
  - 护理按皮革类型区分口径（3365 护理矩阵）：部分皮革（如鹅卵石纹/轻薄/十字纹皮革）适用清洁油膏+保养油膏+防水喷雾；稀有皮革如短吻鳄皮、蜥蜴皮、蟒蛇皮、绒毛麂皮、植鞣皮革等不可用常规清洁保养品，需按专门方式护理；其中**麂皮（绒毛麂皮）店内无法保养，需送专业清洗**。
- **适用条件 / 边界**：
  - 仅"免费清洁和基础护理"，不可等同专业修复、染色、专业防水等高阶服务。
  - 防水喷雾为护理程序之一，仅"轻微防水/有几率做到"，不可承诺"完全防水/可以淋雨"。
  - 门店护理服务的具体收费、时效、预约方式、防水喷雾完整适用皮革清单，课程未写明 → 【待客户补充】（需门店售后 SOP）。
- **如何在对话中自然介绍**：用"任意 Coach 门店免费清洁和基础护理""相当于一直有人帮您照顾它"承接耐久/旧化关切；麂皮等特殊皮革据实引导送专业清洗，不承诺店内可保养。

### 4. Coach 压印服务（吊牌个性化压印）
- **权益内容**：包的吊牌可做个性化压印，很多顾客会刻字母或有纪念意义的日期，强化"专属一只"感；引导顾客在企业微信上联系导购本人代办制作。
- **适用条件 / 边界**：压印的费用、可压内容限制、制作周期，课程未给出 → 【待客户补充】（需压印服务 SOP），导购不宜编造承诺。
- **如何在对话中自然介绍**：用"会更像一只专属的包包"对接都市悦己型/质感自用型的专属与纪念心理，引导企微对接，不承诺具体费用与周期。

### 5. 专业建议与使用场景延展（搭配参考）
- **权益内容**：成交后通过企业微信推送几篇搭配参考，"都是实际背着包包的效果图",给平时怎么搭配做参考，延展使用场景。
- **适用条件**：依托企微触达；属服务延展，非产品参数。
- **如何在对话中自然介绍**：以"稍后在企微给您发几篇实拍搭配参考"承接"买回去会不会不会搭"的隐性顾虑。

### 服务边界红线汇总（勿夸大 / 勿编造）
- 保修：仅"一年保修,范围包括拉链、五金配件等"；"等"字外范围、皮面/人为损坏、起算与凭证要求未说明 → 勿臆测，标【待客户补充】。
- 护理：仅"免费清洁和基础护理"；防水喷雾仅"轻微防水/有几率"；勿等同专业修复/染色/完全防水。麂皮店内无法保养，勿承诺店内护理。
- 退换 / 压印：条件、时效、费用课程均未给出 → 一律标【待客户补充】，勿编造承诺。
- 企微/企业微信为渠道触达工具，非 Coach 品牌专有术语。

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客成交/交付环节据本文档判断"买完之后有没有人管我"：被主动告知企微跟进、保修、免费护理、专属压印时，售后焦虑下降、对成交更安心；礼品选购型据"不合适可退换"消解送礼顾虑；质感自用型/都市悦己型据"免费护理+专属压印"对耐久与专属心理被打动；线上下单顾客据"全国门店一年保修"确认线上线下一体履约。学员若回避服务、只甩政策或夸大承诺，AI 顾客可表现为追问或变冷淡。

### 对话评估引用
评判学员成交后是否把增值服务讲全、讲准、不讲过：是否主动引导加企微并说明用途；保修是否说出"一年+全国门店+线上下单适用"且不夸大为"全包/终身"；礼赠是否给出退换/处理路径；是否承接免费清洁与基础护理且不夸大为专业修复/完全防水；麂皮是否据实引导专业清洗；是否介绍个性化压印强化专属感；是否把系统流程转译为人格化的尊享体验，而非机械念政策。课程未给收费/时效字段处不得编造，标【待客户补充】。
