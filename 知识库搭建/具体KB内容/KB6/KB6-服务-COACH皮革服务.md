# COACH 皮革服务

文档编号：KB6-服务-COACH皮革服务
所属知识库：KB6 门店服务规范
版本：v1.0
更新日期：2026-06-20
数据来源：The Coach Experience(2023-02-27)

## 概述
本文档界定 Coach 门店在 BUILD（打造）环节对顾客提供的"COACH 皮革服务"边界。课程总述（p13）："通过提供独特的 Coach 皮革服务及店铺设计，为顾客打造轻松有趣的空间，激发对品牌和产品的渴望。"课程把门店可提供的皮革服务归为**四类：印字/烫印服务、五金维护、皮革保养与护理、皮革工艺加工**（p13）。

本文档定位是"门店提供什么皮革服务"的服务规范，回答顾客"这包用久了/坏了你们管不管""能不能帮我做点专属的"这类关切，**不是产品材质知识**（皮革本身的材质事实见 [[KB2-正价-皮革与材质知识]]）。供陪练 AI 顾客判断"门店能为我的包做什么"，供评估判断学员是否讲全、讲准服务边界且不夸大、不编造收费与时效。

> 来源说明：课程 p13 为服务网格图，每格配图与小字，**部分小字图中仅部分可辨**。本文档仅沉淀图中可确认的服务类别与课程总述，凡具体收费、时效、预约方式等小字未能辨识者一律标【待客户补充】，不臆测。

## 核心内容

### 一、四类皮革服务总览（课程 p13）
| 服务类别 | 课程定位（p13） | 服务边界要点 |
|---|---|---|
| **印字 / 烫印服务** | 在皮革产品上做印字/烫印的个性化服务 | 与增值服务体系篇"吊牌个性化压印"同源；可压内容/费用/周期【待客户补充】 |
| **五金维护** | 对产品五金配件的维护服务 | 与"全国门店一年保修（含拉链、五金配件等）"口径衔接；具体维护项目与边界【待客户补充】 |
| **皮革保养与护理** | 门店对皮革的清洁与基础护理 | 仅"免费清洁和基础护理"，按皮革类型区分口径，详见下文第二节 |
| **皮革工艺加工** | 对皮革产品做工艺加工 | 课程仅给类别名，具体加工项目与边界图中小字未能辨识 → 【待客户补充】 |

> 课程 p13 小字图中未能完整辨识各格的收费、时效、预约方式与具体作业项目；如需逐字以原始 PPT/门店服务 SOP 为准。

### 二、皮革保养与护理（可对顾客说明的边界）
来源：[[KB6-服务-增值服务体系]] 第 3 节（护理边界，来源 3708/3365），作为 p13"皮革保养与护理"格的可确认细节补充。

- **基础口径**：包可带到任意一家 Coach 门店，门店提供**免费清洁和基础护理**，相当于"一直有人帮您照顾它"。
- **防水喷雾**：护理服务里提供防水喷雾程序，仅"**能有几率做到轻微防水**"，**不可承诺"完全防水/可以淋雨"**。
- **按皮革类型区分护理口径**：
  - 部分皮革（如鹅卵石纹/轻薄/十字纹皮革）适用清洁油膏 + 保养油膏 + 防水喷雾。
  - 稀有皮革（短吻鳄皮、蜥蜴皮、蟒蛇皮、绒毛麂皮、植鞣皮革等）**不可用常规清洁保养品**，需按专门方式护理。
  - **麂皮（绒毛麂皮）店内无法保养，需送专业清洗**——据实引导，不承诺店内可保养。
- **边界红线**：
  - 仅"免费清洁和基础护理"，**不可等同专业修复、染色、专业防水等高阶服务**。
  - 门店护理服务的具体**收费、时效、预约方式、防水喷雾完整适用皮革清单**，课程未写明 → 【待客户补充】（需门店售后/护理 SOP）。

### 三、印字 / 烫印服务（与压印同源的边界）
来源：[[KB6-服务-增值服务体系]] 第 4 节（吊牌个性化压印）。

- **服务内容**：包的吊牌可做个性化压印/烫印，很多顾客会刻字母或有纪念意义的日期，强化"专属一只"感。
- **对接方式**：引导顾客在企业微信上联系导购本人代办制作。
- **边界红线**：压印/烫印的**费用、可压内容限制、制作周期**，课程未给出 → 【待客户补充】（需压印/烫印服务 SOP），不编造承诺。

### 四、五金维护（与保修口径衔接）
来源：[[KB6-服务-交易与服务边界(退换货)]]（保修口径，来源 3708）。

- **可衔接口径**：全国门店提供一年保修，保修范围**包括拉链、五金配件等**；线上下单的产品同样适用全国门店保修。
- **边界红线**："等"字之外是否覆盖皮面、缝线、人为损坏，课程未说明；**不宜夸大为"全包/终身保修"**。五金维护的具体作业项目、是否收费、时效 → 【待客户补充】（需门店售后 SOP）。

### 五、皮革工艺加工
- 课程 p13 仅给出"皮革工艺加工"类别名称，**具体加工项目、适用产品、收费、时效等小字图中未能辨识** → 【待客户补充】（需门店皮革工艺服务 SOP）。
- 红线：在客户补充前，学员**不得编造**任何具体的工艺加工项目或承诺。

### 服务边界红线汇总（勿夸大 / 勿编造）
- 护理：仅"免费清洁和基础护理"；防水喷雾仅"轻微防水/有几率"；勿等同专业修复/染色/完全防水；麂皮店内无法保养。
- 印字/烫印：可做吊牌个性化压印；费用/内容限制/周期未给 → 标【待客户补充】。
- 五金维护：衔接"一年保修含拉链、五金配件等"；勿夸大为全包/终身；具体维护边界【待客户补充】。
- 皮革工艺加工：课程仅给类别名 → 具体内容【待客户补充】，勿编造。

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客在 BUILD/成交后场景据本文档判断"门店能为我的包做什么"：质感自用型/都市悦己型据"免费清洁和基础护理""可做专属压印"被打动耐久与专属心理；关心耐用的顾客据"五金维护+一年保修"安心；问到"能不能完全防水/能不能淋雨"时，期待学员据实说"仅轻微防水"；问麂皮护理时期待被据实引导送专业清洗。学员若把护理夸大为专业修复/完全防水、把保修说成全包/终身，或编造工艺加工项目与收费时效，AI 顾客可追问或表现迟疑。

### 对话评估引用
评判学员是否把门店皮革服务讲全、讲准、不讲过：是否能说出四类服务（印字/烫印、五金维护、皮革保养护理、皮革工艺加工）的存在；护理是否说"免费清洁和基础护理"且不夸大、防水仅说"轻微"、麂皮据实引导专业清洗；五金是否衔接"一年保修含拉链/五金"且不夸大为全包/终身；遇到课程未给的收费/时效/工艺细节是否标注以门店 SOP 为准而非编造。课程未给字段处不得臆测，标【待客户补充】。
