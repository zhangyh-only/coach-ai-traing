## 概述
本文档界定 Coach 门店在 BUILD（打造）环节提供的 **COACH CREATE** 个性化定制体验。课程口号（p14）："**You Create It, We Craft It**"（你来创造，我们来打造）。配文理念（p14）："COACH CREATE 体验概念旨在让顾客感受独一无二的订制体验……充满乐趣的过程也是顾客表达自我个性的绝佳途径。"

COACH CREATE 在课程 p14 分为**四种服务：Personalize it 印记服务 / Customize it 个性服务 / Accessorize it 配饰服务 / Make It To Order 订制服务**。本文档定位是"门店提供什么个性化定制服务"的服务规范，供陪练 AI 顾客判断"我能不能做一只独一无二的包"，供评估判断学员是否能把个性化体验讲出来、且不编造未确认的内容/费用/周期。

> 来源说明：本文档四种服务的名称与配文理念来自课程 p14；各服务的具体可定制内容、收费、制作周期、门店覆盖范围等课程未展开者一律标【待客户补充】，不臆测。

## 核心内容

### 一、COACH CREATE 四种服务（课程 p14）
| 服务 | 中文名（p14） | 课程定位 | 边界要点 |
|---|---|---|---|
| **Personalize it** | 印记服务 | 在产品上做个人印记（印字/压印等），让产品成为"专属一只" | 与 [[KB6-服务-增值服务体系]]"吊牌个性化压印"同源；可压内容/费用/周期【待客户补充】 |
| **Customize it** | 个性服务 | 个性化定制产品 | 具体可定制项目、费用、周期课程未展开 → 【待客户补充】 |
| **Accessorize it** | 配饰服务 | 通过配饰为产品做个性化搭配 | 可选配饰范围、费用课程未展开 → 【待客户补充】 |
| **Make It To Order** | 订制服务 | 按需订制产品 | 可订制范围、起订条件、费用、周期课程未展开 → 【待客户补充】 |

> 课程 p14 仅给出四种服务的名称与整体理念，**未逐项展开**各服务的可定制内容、收费、制作周期与门店覆盖范围；上述具体字段均需门店 COACH CREATE 服务 SOP 补充确认。

### 二、Personalize it 印记服务（可确认的边界细节）
来源：[[KB6-服务-增值服务体系]] 第 4 节（吊牌个性化压印，来源 3708），作为 p14"Personalize it"的可确认细节补充。

- **服务内容**：包的吊牌可做个性化压印，很多顾客会刻字母或有纪念意义的日期，强化"专属一只"感。
- **对接方式**：引导顾客在企业微信上联系导购本人代办制作。
- **边界红线**：压印的**费用、可压内容限制、制作周期**课程未给出 → 【待客户补充】（需压印服务 SOP），不编造承诺。

### 三、体验理念与体验场景
- **核心理念（p14）**：COACH CREATE 让顾客感受**独一无二的订制体验**，充满乐趣的过程也是顾客**表达自我个性**的绝佳途径。对接"想要一只和别人不一样的包""有纪念意义"的专属与悦己心理。
- **体验场景示例（p14）**：课程以 **CHENGDU IFS（成都 IFS）门店**作为 COACH CREATE 直播体验空间示例。
- **边界**：除成都 IFS 直播体验空间外，**哪些门店提供 COACH CREATE、各项服务的具体操作流程**课程未说明 → 【待客户补充】（需门店服务覆盖范围 SOP）。

### 服务边界红线汇总（勿夸大 / 勿编造）
- 四种服务（Personalize/Customize/Accessorize/Make It To Order）名称与理念来自课程 p14，可介绍其存在与"独一无二/表达自我"的体验价值。
- 各服务的**可定制内容、费用、制作周期、门店覆盖范围**课程均未展开 → 一律标【待客户补充】，**不编造**具体项目、价格与时效。
- 成都 IFS 仅作为体验场景示例提及，不可据此推断其他门店的服务能力。

## 在陪练系统中的应用
### 陪练对话引用
AI 顾客在 BUILD 环节据本文档判断"我能不能做一只独一无二的包"：都市悦己型/质感自用型据"COACH CREATE 让你感受独一无二、表达自我个性"被打动专属与悦己心理；想送有纪念意义礼物的顾客据"印记服务可刻字母/纪念日期"被承接。顾客可能追问"具体怎么定制/多少钱/多久做好/哪些门店能做"，期待学员据实说"具体内容/费用/周期以门店 COACH CREATE 服务为准"而非编造。学员若凭空报出定制项目、价格或周期，AI 顾客可追问细节或表现迟疑。

### 对话评估引用
评判学员是否把 COACH CREATE 讲出来、讲准：是否能介绍四种服务的存在与"独一无二/表达自我"的体验价值；印记服务是否能落到"吊牌压印可刻字母/纪念日期"并引导企微对接；遇到课程未给的可定制内容/费用/周期/门店覆盖是否标注以门店 SOP 为准而非编造；成都 IFS 是否仅作体验场景示例提及而非泛化为所有门店能力。课程未给字段处不得臆测，标【待客户补充】。
